# حدود التشغيل المعروفة قبل النشر العام

1. **GitHub Pages ليس خادم ألعاب:** هو استضافة ثابتة فقط. المزامنة الحقيقية بين اللاعبين، الغرف المركزية، المحافظ، الصلاحيات، سجل النادي والتحديات الرسمية تحتاج خادم Laravel وقاعدة بيانات.
2. **الإعلانات الحالية اختبارية افتراضيًا:** يجب إدخال معرّفات AdMob الإنتاجية واعتماد الحساب قبل النشر التجاري.
3. **التوقيع والنشر:** AAB الإنتاجي يحتاج مفتاح Google Play، وiOS يحتاج شهادات Apple وProvisioning Profiles.
4. **التحقق المنفذ في هذه الحزمة:** تم فحص المصدر والعقود ومحركات PHP وبنية ملفات Flutter، لكن بيئة إعداد الحزمة لم تتضمن Flutter/Dart SDK أو Composer؛ لذلك لم يُنفذ فيها `flutter analyze` أو بناء APK/Web أو PHPUnit. ملفات GitHub Actions مجهّزة لتنفيذ هذه المراحل في بيئة CI.
5. **لا يوجد برنامج برمجي يمكن ضمان خلوه مطلقًا من جميع العيوب:** الاختبارات الحالية تقلل المخاطر وتغطي العقود الرئيسية، ويجب تنفيذ فحص CI واختبار قبول يدوي متعدد الأجهزة قبل الإطلاق العام.
