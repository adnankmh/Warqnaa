# Definition of Done

A feature is complete only when all applicable requirements below are true:

1. Behavior is enforced by the server for security-sensitive actions.
2. Automated tests cover the success path and important failure paths.
3. Flutter analysis and tests pass.
4. Laravel tests pass.
5. Arabic RTL and all enabled languages are supported without hard-coded user-facing text.
6. Loading, empty, offline and error states are implemented.
7. Weak-network retries do not duplicate wallet, reward or match results.
8. Accessibility font scaling and portrait/landscape layouts are verified.
9. Analytics and audit events are added when operationally necessary.
10. Documentation and changelog are updated.
11. The feature is tested on Web and Android.
12. A reviewer approves the pull request before merge.
