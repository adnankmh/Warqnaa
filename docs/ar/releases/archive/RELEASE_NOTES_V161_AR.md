# ملاحظات إصدار Warqna v161

## الاستقرار والتشغيل

- تشغيل Flutter يرسم أول إطار قبل تهيئة AdMob وأي SDK اختياري.
- أخطاء الإعلانات لا تغلق APK ولا تمنع شاشة الدخول.
- إصلاح خطأ Dart مزدوج `else` في معالجة ICE الصوتية.
- إعادة بناء اتصال WebRTC المتعطل بدل الاعتماد على استدعاء غير مضمون بين إصدارات المكتبة.
- Android Manifest يحصل آليًا على الإنترنت والميكروفون وAdMob App ID وSDK 36 وminSdk 24 أثناء البناء.

## الملفات الشخصية والدول

- قائمة 248 دولة/منطقة مع الاسم العربي والإنجليزي والعلم.
- ظهور الدولة والعلم في البروفايل وبيانات اللاعبين داخل الغرف والصوت.
- تقدم المستوى، المتبقي للمستوى التالي، نقاط الجولات، نقاط المسابقات ونقاط النادي.

## النقاط

- احتساب Server-authoritative بعد كل جولة مع مفتاح idempotency يمنع التكرار.
- الباشا ×2 مرة واحدة، والمسرعات تتراكم بصورة صحيحة دون تربيع المضاعف.
- المسابقة العادية ×2، والمدعومة أو الموسمية ×3.
- نقاط صدارة المسابقات: 1000 للبطل، 600 للوصيف، 350 لنصف النهائي، 150 لربع النهائي، و35 للمشاركة المبكرة.
- نقاط النادي ترتفع مع الفوز، درجة الدوري، ولعب أعضاء النادي معًا ×2.

## النوادي

- المالك يحدد للمشرف صلاحيات قبول الطلبات، الطرد، إنشاء المسابقات، إدارة الدردشة ونشر الإعلانات.
- إعلانات نادي مثبتة وقابلة للحذف.
- منشئ مسابقات خاص بالنادي من مرحلة واحدة حتى أربع مراحل.

## المتجر والإدارة

- رفع صورة طاولة أو ظهر ورق أو غلاف بروفايل من لوحة الإدارة.
- تظهر الصورة في المعاينة، المتجر، والغرفة بعد التفعيل.
- مؤثرات فوز وشارات وباقات إيموجي إضافية بأسعار ومستويات متعددة.
- إيموجي أكبر ومقسمة إلى تبويبات المرح، المشاعر، الاحتراف، الأسطوري والرياضة.
- كل حساب مدير يحصل على 3650 يوم باشا عند الترحيل أو إنشاء قاعدة البيانات.

## تسجيل الدخول

- مسار OAuth مركزي لـGoogle وFacebook وApple.
- فتح صفحة المزود، polling آمن لحالة الجلسة، وإصدار Sanctum token مرة واحدة.
- لا تظهر أزرار مزيفة: عند عدم إضافة المفاتيح يعيد الخادم رسالة إعداد واضحة.
