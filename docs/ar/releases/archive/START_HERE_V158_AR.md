# ابدأ من هنا — Warqna v158

هذه حزمة مصدر كاملة تضم Flutter وLaravel وGitHub Actions، وليست ملفات ترقيع منفصلة.

## الإصلاحات الجذرية

- استبدال فحص `grep -A2` الهش بأداة `tools/verify_flutter_lock.py` التي تقرأ كتلة الحزمة كاملة من `pubspec.lock` وتتحقق من الإصدار الحقيقي مهما تغيّر ترتيب بيانات Pub.
- الإبقاء على تثبيت `google_mobile_ads 7.0.0` و`flutter_webrtc 1.4.0` مع اختبار ذاتي للمحلل قبل البناء.
- الحفاظ على رمز SQLite الخاص `:memory:` بدل تحويله خطأ إلى مسار ملف فعلي، مع دعم SQLite URI أيضًا.
- إصلاح تطبيع صيغ الورق العربية والرمزية والإنجليزية دون استبدالات حرفية متداخلة تفسد أسماء الأنواع.
- جعل محرك الطرنيب يدعم الحالة الحديثة والحالات القديمة مع مزامنة الدور والمزايدة والحكم واليد واللمّة والنقاط.
- توفير مسارات PWA صريحة ومسارات توافق عامة للإصدارات القديمة من Mobile API.
- تحديث اختبارات Laravel القديمة لتطابق عقد المنتج الحالي: 12 لعبة مختارة، 50 طاولة، 40 ظهر ورق، وعدم إعادة الألعاب المحذوفة.
- توحيد الإصدار إلى `1.58.0+158` في Flutter وLaravel ومسارات Web وAndroid وiOS.

## الفحص المحلي

على Windows:

```bat
CHECK_V158_WINDOWS.bat
```

على Linux/macOS:

```bash
./check-v158.sh
```

بعد الرفع، سيشغّل GitHub Actions الاختبارات الكاملة، ثم يمكن تشغيل `Build Android APK and AAB` يدويًا لاستخراج APK وAAB.
