كانت 11 وحدة تحكم في Mobile API والصفحات القانونية تكتب `extends Controller`، بينما الملف الأساسي التالي غير موجود داخل الحزمة:

`backend-laravel/app/Http/Controllers/Controller.php`

لذلك كانت مسارات الصحة، إعدادات التطبيق، كتالوج الألعاب، التصدير، الجلسات، البلاغات وتحويل التوكنز تعيد خطأ خادم بدل استجابة HTTP صحيحة.

اختباران قديمان كانا يشترطان حصول كل لاعب على ورقتين قويتين على الأقل. هذا الشرط يعتمد على الحظ ويتعارض مع التوزيع العشوائي العادل. أصبح الفحص يتأكد من:

- 13 ورقة لكل لاعب.
- 52 ورقة إجمالًا.
- عدم وجود أي ورقة مكررة.
- عدم تعديل أو تقوية يد لاعب بعينه.
