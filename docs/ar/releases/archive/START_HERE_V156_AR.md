# ابدأ من هنا — Warqna v156

هذه حزمة كاملة تضم Flutter وLaravel وGitHub Actions، وليست ملفات ترقيع منفصلة.

## الإصلاحات

- منع انهيار محرك الطرنيب عند اكتمال آخر لَمّة في حالة اختبار أو استعادة جلسة لا تحتوي بيانات مزايدة كاملة.
- الإبقاء على آخر لَمّة ظاهرة حتى يبدأ اللاعب الفائز اللَمّة التالية.
- تحويل حقل فئة عناصر المتجر من Enum مغلق إلى String قابل للتوسعة قبل ترحيل `000145`.
- تصنيف الثيمات داخل فئة `effect` المتوافقة مع منطق التفعيل.
- دعم فئة `profile_cover` في Laravel والمتجر ولوحة الإدارة.
- إضافة اتصال PostgreSQL إلى `config/database.php` ليتطابق Laravel مع Docker Production.
- توحيد الإصدار إلى `1.56.0+156`.
