# ملاحظات إصدار Warqna v153

## الأساس الإنتاجي

- إدارة Feature Flags.
- إدارة إصدارات Web/Android/iOS والتحديث الإجباري.
- Health/Ready endpoints.
- Request IDs وSecurity Headers.
- Rate Limiting.
- Maintenance Mode.
- Admin Audit Logs.
- Docker Production Stack.
- CI للـBackend والتحقق الأمني.
- Workflow لنشر Staging وProduction.

## الحساب

- استعادة كلمة المرور.
- صفحة ويب آمنة لتعيين كلمة مرور جديدة.
- تأكيد البريد.
- إدارة الجلسات والأجهزة.
- تصدير بيانات الحساب.
- طلب حذف مع فترة سماح وإلغاء الطلب.
- تعطيل الدخول المحلي تلقائيًا في Production Build.

## السلامة

- بلاغات مستخدم/غرفة/رسالة.
- لوحة مراجعة إدارية.
- إجراءات حل/حظر.
- صفحات الخصوصية والشروط وقواعد المجتمع والمنافسات والدعم.

## الألعاب والصوت

- الحفاظ على كل محركات وواجهات v152.
- الحفاظ على خيار اللعبة العادية أو الصوتية.
- WebRTC signaling وPresence وMute/Deafen.
- إعداد TURN/STUN من البيئة.

## النشر

- Flutter Web مع وضع Preview عند غياب API ووضع Production عند وجوده.
- APK/AAB مع AdMob 7 وFlutter مثبت الإصدار.
- توقيع Google Play اختياري من GitHub Secrets.
- SHA-256 لملفات Android الناتجة.
