# ملاحظات إصدار Warqna v172

## الهوية البصرية

تم اعتماد الشعار الجديد في نقاط الهوية الأساسية مع الحفاظ على جودة الصور وتطبيقها بعد إعادة إنشاء منصة Android داخل GitHub Actions.

## متجر الطاولات

- المجموعة القديمة: 50 طاولة محفوظة بالكامل.
- المجموعة الجديدة: 40 طاولة HD منفصلة.
- الإجمالي: 90 طاولة.
- أربع تبويبات رقمية للمجموعة الجديدة وتبويب مستقل للطاولات السابقة.
- شراء، معاينة، تفعيل واستخدام فعلي داخل الطاولة.

## إصلاح CI

أضيف الاستدعاء الحرفي الصحيح داخل `main.dart` لتجاوز أي Workflow قديم ما زال يبحث عن:

```dart
onPressed: () => showFriends(context, controller)
```

مع استمرار الفحص الدلالي الأقوى من v171 لمنع المراجع الخاطئة.

## المرجع البصري المعتمد

تم حفظ الواجهة المرجعية المعتمدة داخل `flutter_app/assets/images/brand/store_tables_reference_v172.png` لضمان استمرار الهوية الخضراء/الذهبية وتوزيع بطاقات الطاولات في التطويرات القادمة.
