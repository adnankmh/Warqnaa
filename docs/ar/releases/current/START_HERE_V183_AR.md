# ابدأ من هنا — Warqnaa V0.3.2 / v183

الإصدار التقني: **0.3.2+183**.

يتضمن هذا الإصدار واجهة ويب مكتبية متجاوبة، معاينات كاملة للطاولات والعناصر، سبعة مسرعات ملونة، تنظيف تبويبات الطاولات المؤقتة، توسيع المصمم الشامل، وتطوير محركات طرنيب 400 والهاند والبناكل وتركس والبلوت في الخادم والوضع المحلي.

## التشغيل
1. شغّل `CHECK_WARQNA_WINDOWS.bat`.
2. بعد نجاح الفحص شغّل `START_WARQNA_WINDOWS.bat`.
3. للنشر اضبط Laravel و`WARQNA_API_URL` ومفاتيح الخدمات الخارجية.
4. لا تنشر قبل نجاح GitHub Actions للويب وAndroid وiOS والخادم.
