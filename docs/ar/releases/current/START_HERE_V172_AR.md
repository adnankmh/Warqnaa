# ابدأ من هنا — Warqna v172

الإصدار: **1.72.0+172**  
الأساس: **v171**

## هدف الإصدار

اعتماد شعار ورقنا الجديد في واجهات التطبيق، وإضافة مجموعة مرجعية كاملة من 40 طاولة عالية الجودة إلى المتجر **دون استبدال أو حذف الطاولات الخمسين السابقة**. أصبح إجمالي الطاولات 90 طاولة.

## ما أضيف

- شعار `ورقنا / Warqna` في شاشة البداية، تسجيل الدخول، الشريط العلوي، المتجر، الويب وأيقونات Android وPlay Store.
- تبويب طاولات منظم إلى: الجديدة 1–10، 11–20، 21–30، 31–40، والطاولات السابقة.
- صورة HD مستقلة، اسم وسعر ومعاينة وتفعيل لكل طاولة جديدة.
- استخدام صورة الطاولة المفعلة فعلًا داخل غرفة اللعب.
- الحفاظ على 50 طاولة قديمة وإضافة 40 جديدة، بإجمالي 90.
- توافق مزدوج مع فحص GitHub القديم النصي وعقد v171 الدلالي.

## الفحص المحلي

على Windows شغّل `CHECK_WARQNA_WINDOWS.bat`.

على Linux/macOS شغّل:

```bash
bash scripts/unix/current/check-v172.sh
```
