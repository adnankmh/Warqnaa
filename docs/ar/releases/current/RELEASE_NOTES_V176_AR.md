# ملاحظات إصدار Warqna v176

## إصلاحات Flutter Analyzer

- إزالة الحقل غير المستخدم `_openingRoomRouteV174`.
- معالجة استخدام `BuildContext` بعد العمليات غير المتزامنة عبر أخذ سياق جديد من `warqnaNavigatorKey` والتحقق من `mounted`.
- إزالة interpolation غير الضروري في `v175_release.dart`.
- إبقاء مسار الانتقال المباشر إلى الغرف والدعوات دون كسر عقود V174/V175.

## الحزمة اليومية والمخزون

- نافذة فتح متحركة مع صندوق ملكي، مؤثرات دوران ونبض واهتزاز، ثم كشف الجائزة حسب الندرة.
- الجوائز المؤقتة تنشئ عنصراً في المتجر ومقتنى خادمياً مرتبطاً بالحساب.
- دعم انتهاء صلاحية ألوان الاسم والدردشة، مسرعات XP والطاولات المؤقتة.
- تبويب **مقتنياتي** يعرض العناصر الفعالة فقط مع الوقت المتبقي.
- سجل مختصر لجوائز الحزم يوضح العناصر الفعالة والمنتهية.
- التوكنز والتذاكر تضاف مباشرة إلى الرصيد دون إنشاء مقتنيات وهمية.
