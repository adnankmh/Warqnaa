# ابدأ من هنا — Warqnaa V0.3.1 / v182

الإصدار التقني: **0.3.1+182**.

هذا الإصدار يضيف دولاب الحظ الملكي، ويطوّر صناديق الجوائز بحسب نتيجة ونوع المباراة، ويصحّح الغياب والانقطاع والخروج اليدوي، ويفتح المصمم الشامل والمزامنة لحساب **Adnan** فقط، ويربط إيرادات المتجر بحساب الإدارة، ويثبت مستوى المدير على 99.

## التشغيل السريع

1. استخدم الملف الكامل، أو فك الجزأين داخل المجلد نفسه.
2. لا تخلط ملفات هذا الإصدار مع نسخة أقدم.
3. على Windows شغّل `CHECK_WARQNA_WINDOWS.bat` أولًا.
4. بعد نجاح الفحص شغّل `START_WARQNA_WINDOWS.bat`.
5. للنشر الحقيقي اضبط خادم Laravel وقاعدة البيانات و`WARQNA_API_URL` ثم شغّل GitHub Actions.

راجع `docs/INSTALL_V0.3.1_AR.md` و`CHANGELOG_V0.3.1_AR.md` قبل النشر.
