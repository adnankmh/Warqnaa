# ابدأ من هنا — Warqnaa V0.3.3 / v184

الإصدار التقني: **0.3.3+184**.

يتضمن هذا الإصدار دولاب حظ من 10 خانات، مؤشرًا أكبر، إصلاحات CI وLaravel، صور المسرعات الجديدة المضغوطة، مع الاحتفاظ بواجهة الويب المكتبية ومعاينات الطاولات والمصمم الشامل ومحركات الألعاب المطورة.

## التشغيل
1. شغّل `CHECK_WARQNA_WINDOWS.bat`.
2. بعد نجاح الفحص شغّل `START_WARQNA_WINDOWS.bat`.
3. للنشر اضبط Laravel و`WARQNA_API_URL` ومفاتيح الخدمات الخارجية.
4. لا تنشر قبل نجاح GitHub Actions للويب وAndroid وiOS والخادم.
