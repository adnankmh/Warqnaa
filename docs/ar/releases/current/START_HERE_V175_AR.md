# ابدأ من هنا — Warqna v175

الإصدار **1.75.0+175**. هذه النسخة تدمج جدول XP الرسمي من ملف Excel للمستويات 1–100، تسجيل الويب المحلي عند تعذر الخادم، توجيه دعوات الغرف مباشرة، تثبيت اتجاه الشاشة حسب اختيار المستخدم، تحديات وحزم خادمية، صورة الباشا الحمراء الكاملة، والمصمم الشامل.

نفّذ `php artisan migrate --force`، ثم ارفع المشروع وشغّل جميع GitHub Actions. اضبط عنوان Laravel الحقيقي في `WARQNA_API_URL` للميزات المتصلة.
