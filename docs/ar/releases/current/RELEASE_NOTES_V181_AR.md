# ملاحظات إصدار Warqna V0.2.5 — البناء 181

## اللعب والعدالة
- إضافة `FairDealBalancer` في Laravel وFlutter بعد الخلط العشوائي الآمن.
- الحفاظ على 52 ورقة فريدة وأحجام الأيدي الصحيحة، مع تقليل الأيدي شديدة الضعف دون تفضيل لاعب أو حساب.
- تطبيق الموازنة على الطرنيب والمحرك المحلي والمحركات العالمية.

## الغياب والمغادرة
- عداد مستقل للغياب وآخر للمغادرة اليدوية.
- إخراج مؤقت بعد 3 لفات غياب مع السماح بالعودة.
- منع العودة إلى الجلسة نفسها فقط بعد 3 مغادرات يدوية.

## التحديات والمستويات
- مسارات 10/12/15 مرحلة، 5 محاولات، ولعبة ثابتة حتى نهاية المسار.
- تفضيل منافس متصل خلال آخر 3 دقائق، ثم حساب حقيقي عشوائي أو بوت احتياطي عند عدم توفره.
- جوائز محددة السقف: توكنز لا تتجاوز 1000، باشا، طاولات، ألوان، تذاكر، مسرعات وصناديق.
- ربط كل نتيجة بنوع اللعبة ومنع تكرار نتائج الطلبات السابقة عبر سجل معرّفات آمن.
- مكافآت انتقال مستوى محفوظة لمرة واحدة فقط.

## الإدارة والمتجر والواجهة
- المصمم الشامل محمي في الواجهة والخادم لحساب المدير الرئيسي `Adnan`.
- إرسال 200 توكن أو طلب صداقة لأي مستخدم موجود في قاعدة البيانات.
- تبويبات متجر مترجمة، أزرار 3D، إزالة عنوان المتجر المكرر، وإعادة تسمية زر المتجر في التنقل السفلي.
- تكبير افتراضي منضبط للخطوط والأزرار، مع شريط تحكم ثابت للثيم واللغة والخط والاتجاه.
