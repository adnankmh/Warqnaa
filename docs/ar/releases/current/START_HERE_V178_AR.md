# ابدأ من هنا — Warqna v178

الإصدار **0.2.2+178**. تعالج هذه النسخة تحذيرات Flutter Analyzer التي أوقفت CI، وتضيف فتحاً متحركاً وفخماً للحزمة اليومية، وإضافة الجوائز المؤقتة مباشرة إلى مخزون الخادم وتبويب **مقتنياتي** في المتجر مع تاريخ انتهاء وعدّاد صلاحية وإزالة تلقائية بعد الانتهاء.

نفّذ `php artisan migrate --force`، واضبط عنوان Laravel الحقيقي في `WARQNA_API_URL`، ثم شغّل جميع GitHub Actions قبل النشر.
