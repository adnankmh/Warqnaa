# ابدأ من هنا — Warqna V0.2.2 (البناء 178)

هذا التحديث هو **ملف Patch فقط** ويُطبَّق فوق مشروع **V0.2 / V0.2.1**. يعالج الاقتصاد والشراء، ويفعّل الغرف المفتوحة، وصلاحيات الإدارة والمجموعات، ويطوّر محركات اللعب الجماعي، ويضيف تخصيص ألعاب الصفحة الرئيسية والثيمات المدفوعة وتحسينات البروفايل والمتجر والدردشة والطاولات.

## خطوات التطبيق

1. فك ملف التحديث في جذر المشروع ووافق على استبدال الملفات.
2. شغّل داخل `backend-laravel`:
   - `composer install` عند الحاجة.
   - `php artisan migrate --force`.
   - `php artisan optimize:clear`.
   - `php artisan test`.
3. شغّل داخل `flutter_app`:
   - `flutter pub get`.
   - `flutter analyze`.
   - اختبارات وبناء المنصة المطلوبة.
4. نفّذ جميع GitHub Actions قبل النشر.

الخادم هو المرجع النهائي للرصيد والشراء والتحويل والصلاحيات والغرف العامة وسجل النادي.
