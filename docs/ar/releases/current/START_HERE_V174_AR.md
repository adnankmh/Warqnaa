# ابدأ من هنا — Warqna v174

الإصدار: **1.74.0+174**  
الأساس: **v173**  
اسم الإصدار: **Direct Invite, Stable Orientation & Authoritative XP**

## أهم تعديلات الإصدار

- إصلاح أخطاء Flutter Analyzer التي أوقفت GitHub Actions، بما فيها تسجيل الدخول الاجتماعي، الأكواد غير المستخدمة، واستخدام `notifyListeners` خارج `ChangeNotifier`.
- فتح غرفة اللعبة مباشرة بعد إنشائها.
- فتح اللعبة المدعو إليها اللاعب مباشرة عند الضغط على إشعار الدعوة، سواء كان التطبيق مفتوحاً أو في الخلفية أو تم تشغيله من الإشعار.
- تثبيت اتجاه التطبيق وفق اختيار اللاعب؛ الوضع الافتراضي طولي ولا يتحول إلى أفقي تلقائياً عند فتح اللعبة.
- اعتماد منحنى نقاط الخبرة الجديد للمستويات 40–100 على Flutter وLaravel.
- إظهار نقاط الخبرة المكتسبة بعد كل جولة فوراً، وتحديث عدادات XP ونقاط الجولة والمنافسات والمجموعة لكل لاعب حقيقي، مع استثناء البوتات.
- الحفاظ على جميع ميزات v173، ومنها 140 طاولة، 14 لون باشا، تذاكر المنافسات، الحزمة اليومية، الإعلانات والمصمم الشامل.
