# قائمة إطلاق Warqna العالمية

## قبل Staging

- [ ] نطاقات منفصلة للواجهة والـAPI.
- [ ] PostgreSQL وRedis يعملان.
- [ ] HTTPS صالح.
- [ ] SMTP يعمل لاستعادة كلمة المرور وتأكيد البريد.
- [ ] TURN يعمل عبر UDP وTCP.
- [ ] `warqna:production-check --strict` ناجح.
- [ ] النسخ الاحتياطي والاستعادة مجرّبان.

## اختبارات اللعب

- [ ] كل لعبة تعمل من البداية حتى النتيجة.
- [ ] أربع أجهزة حقيقية في مباراة واحدة.
- [ ] الانقطاع والعودة.
- [ ] انتهاء مؤقت الدور.
- [ ] وضع الغائب.
- [ ] الغرفة العادية والصوتية.
- [ ] دعوات الأصدقاء والمقاعد.
- [ ] منع اللعبتين المتزامنتين.
- [ ] المنافسات من 1 إلى 4 مراحل.
- [ ] المجموعات والأعضاء والحضور.

## المتجر والاقتصاد

- [ ] تأكيد قبل الشراء.
- [ ] الخصم مرة واحدة فقط.
- [ ] انتقال الإيراد إلى محفظة النظام.
- [ ] تحويل التوكنز وعمولة 10%.
- [ ] استرجاع السجل بعد إعادة الدخول.
- [ ] اختبار التزامن ومنع Double Spend.

## الحساب والخصوصية

- [ ] التسجيل والدخول.
