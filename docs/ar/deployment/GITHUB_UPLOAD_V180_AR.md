# رفع وبناء Warqnaa V0.3 / v180 عبر GitHub

1. فك الجزأين داخل مجلد واحد وارفع المشروع كاملًا، بما في ذلك `.github/workflows`.
2. شغّل **Production Release Check** ثم **Backend CI**.
3. شغّل Flutter Web أو Android أو iOS بحسب منصة النشر.
4. اضبط `WARQNA_API_URL` على خادم Laravel الحقيقي لتفعيل المزامنة المركزية.
5. أضف إعدادات قاعدة البيانات وCORS والكاش والجلسات والبريد إلى بيئة الخادم.
6. أضف معرّفات AdMob الإنتاجية وأسرار توقيع Android/iOS قبل النشر في المتاجر.
7. لا تعتبر وضع GitHub Pages المحلي بديلًا عن الخادم للغرف والمحافظ والنوادي الرسمية.
