# تقرير إصلاح أخطاء V174

تمت معالجة الأخطاء المرفقة في مسارين: Flutter وLaravel.

## Flutter

- تعريف تسجيل الدخول الاجتماعي بدل استدعاء دالة غير موجودة.
- إدخال دوال الحساب المحلي في مسار الدخول والتسجيل والاستعادة الفعلي.
- استبدال الوصول الخارجي المحظور إلى `notifyListeners` بدالة عامة داخل المتحكم.
- حذف الفرع الميت في واجهة الدخول.
- إضافة دخول وإنشاء حساب أوفلاين بصورة صريحة.
- تثبيت الاتجاه حسب اختيار اللاعب.
- فتح الغرفة مباشرة بعد الإنشاء أو الانضمام أو الضغط على الإشعار.
- إظهار نتائج XP لكل لاعب حقيقي في الجولة.

## Laravel

- تحديث العدد المتوقع للطاولات إلى 140.
- تصحيح تمثيل تاريخ الحزمة اليومية.
- جعل إنشاء لعبة الاختبار آمناً من تكرار المفتاح.
- إضافة لقطة تقدم معتمدة من الخادم بعد الجولة.
- إضافة اختبارات خاصة بمنحنى XP الجديد وتاريخ الحزمة وفرادة كتالوج الطاولات.
