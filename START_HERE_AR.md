# ابدأ من هنا — Warqna V0.2.5

هذه حزمة المصدر الكاملة للإصدار **V0.2.5 / build 181**، وتشمل مشروع Laravel ومشروع Flutter وملفات الاختبار والبناء والتوثيق.

## التشغيل السريع على Windows

1. فك الضغط في مسار قصير، مثل:
   `C:\Warqna-V0.2.5`
2. ثبّت PHP 8.2+ وComposer وNode.js LTS وFlutter المتوافق مع `pubspec.yaml`.
3. من مجلد `backend-laravel` شغّل لأول مرة:
   `setup-windows.bat`
4. بعد نجاح الإعداد شغّل الخادم:
   `start-windows.bat`
5. من مجلد `flutter_app` نفّذ:
   `flutter pub get`
6. شغّل نسخة الويب عبر:
   `RUN_FLUTTER_WEB.bat`
   أو استخدم `flutter run -d chrome` مع عنوان API المناسب.

## الفحص قبل التشغيل أو الرفع

من جذر المشروع شغّل:
`CHECK_WARQNA_WINDOWS.bat`

وتوجد نتيجة الفحص الحالية في:
`docs/ar/validation/current/VALIDATION_RESULTS_V181.txt`

## ملاحظات مهمة

- لا تشغّل `setup-windows.bat` مرة أخرى بعد إدخال بيانات حقيقية إذا كان ينفذ إعادة تهيئة لقاعدة البيانات.
- عند تشغيل التطبيق على هاتف فعلي، استخدم عنوان IP للكمبيوتر بدل `127.0.0.1`، وشغّل Laravel على `0.0.0.0`.
- عند النشر العام، استخدم خادم HTTPS حقيقي، وغيّر بيانات المدير والحسابات التجريبية، واستبدل معرّفات الإعلانات التجريبية بمعرّفاتك الإنتاجية.
- مجلدا Android وiOS يمكن توليدهما من Flutter عند البناء إذا لم يكونا موجودين في المصدر.
- الحسابات التجريبية موثقة في:
  `docs/ar/product/DEMO_ACCOUNTS_V025_AR.md`
