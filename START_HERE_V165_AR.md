# ابدأ من هنا — Warqna v165

هذه النسخة تعالج انهيار APK قبل فتح واجهة التطبيق الناتج عن WorkManager / WorkDatabase كما ظهر في سجل Logcat.

## الرفع

1. افتح GitHub Desktop.
2. نفذ Fetch ثم Pull.
3. إذا ظهر Merge معلّق اختر Abort merge.
4. أبقِ مجلد `.git` فقط داخل المستودع.
5. انسخ محتويات مجلد v165 الداخلي إلى جذر المستودع.
6. شغّل `CHECK_V165_WINDOWS.bat`.
7. ارفع Commit باسم:

```text
Warqna v165 Android WorkManager boot fix
```

بعد نجاح Actions حمّل APK من Artifact باسم `warqna-v165-android` وثبّت ملف `warqna-v165-safe.apk`.
