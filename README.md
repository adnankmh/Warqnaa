# Warqnaa V0.3.1 — Lucky Wheel & Rewards Release

الإصدار الكامل الموحّد من **Warqnaa — ورقنا**، رقم الإصدار التقني **0.3.1+182**.

يتكوّن المشروع من:

- `flutter_app/`: تطبيق Flutter للويب وAndroid وiOS.
- `backend-laravel/`: خادم Laravel للغرف والحسابات والمتجر والنوادي والمكافآت والإدارة.
- `.github/workflows/`: فحوص وبناء Web وAPK وAAB وiOS.
- `tools/`: عقود رجوعية وفحوص المصدر والإصدار.

أهم إضافات V0.3.1: دولاب حظ من 6 جوائز، صناديق مرتبطة بنتيجة المباراة، غياب دائم اختياري مع إخراج بعد ثلاث لفات من الخمول الحقيقي، عودة إلى المقعد دون استهلاك الخروج اليدوي، مصمم شامل ومزامنة لحساب Adnan فقط، إيرادات متجر ذرّية لحساب الإدارة، إدارة نوادٍ محسنة، اتجاه لعب عكس عقارب الساعة، وثيم فاتح ومعاينات صور محسنة.
