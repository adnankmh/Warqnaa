# Warqnaa V0.3 Professional Release

الإصدار الكامل الموحّد من **Warqnaa — ورقنا**، رقم الإصدار التقني **0.3.0+180**.

يتكوّن المشروع من:

- `flutter_app/`: تطبيق Flutter للويب، Android وiOS.
- `backend-laravel/`: خادم Laravel للغرف الحقيقية، الحسابات، المتجر، النوادي، التحديات والإدارة.
- `.github/workflows/`: بناء واختبار Web وAPK وAAB وiOS عبر GitHub Actions.
- `tools/`: فحوص الإصدار والعقود والأمان وجودة المصدر.

أهم إضافات V0.3 تشمل إصلاح الشراء، الغرف المفتوحة، تخصيص ألعاب الرئيسية، صلاحيات وسجل النوادي، تحديات المراحل، مكافآت المستويات، الإعلانات المكافِئة التجريبية، صناديق الجوائز، تسعة ثيمات أساسية، المصمم المحمي لحساب Adnan، وتحديثات محركات الهاند والبناكل ومنع كشف الأوراق القادمة.

ابدأ من `START_HERE_AR.md` ثم راجع `docs/INSTALL_V0.3_AR.md` و`VALIDATION_V0.3.txt`.
