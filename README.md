# Warqna v173

نسخة إنتاجية موسعة من لعبة **ورقنا** مبنية فوق v172 مع الحفاظ على جميع الميزات والطاولات السابقة.

- الإصدار: `1.73.0+173`
- نمط التشغيل: متصل بالخادم فقط؛ لا تسجيل دخول محلي ولا لعب دون إنترنت.
- طربوش الباشا: 14 لوناً مستقلاً مع تطبيق اللون على هوية الباشا.
- الطاولات: 50 طاولة HD جديدة، ليصبح الإجمالي 140 طاولة مع المجموعات السابقة.
- المنافسات: تذاكر، رسوم دخول، مقاعد عشوائية، قبول تلقائي، جوائز وتحديات يومية/أسبوعية/موسمية.
- المتجر: 14 فئة تذاكر بخصم شراء 10% وحزمة يومية مجانية محدودة المدة.
- الإدارة: مصمم شامل للمدير مع CRUD لعناصر وإعدادات التطبيق.
- الإعلانات: Rewarded Ads مهيأة بأرقام الاختبار افتراضياً ومتغيرات إنتاج منفصلة.
