# Warqna v174

نسخة احترافية محدثة من لعبة **ورقنا** مبنية فوق v173 مع الحفاظ على جميع الميزات والطاولات السابقة.

- الإصدار: `1.74.0+174`
- التشغيل: متصل بالخادم فقط؛ لا تسجيل دخول محلي ولا لعب دون إنترنت.
- الدخول المباشر: فتح غرفة اللعبة فور إنشائها، وفتح الغرفة المدعو إليها اللاعب من الإشعار في حالات التطبيق المفتوح والخلفية والتشغيل البارد.
- اتجاه الشاشة: طولي افتراضياً، ولا يتحول إلى الأفقي إلا باختيار اللاعب مع حفظ الاختيار.
- نقاط الخبرة: منحنى جديد للمستويات 40–100 مطبق بالتطابق في Flutter وLaravel.
- نقاط الجولة: منح خادمي موثوق لكل لاعب حقيقي، ظهور واضح بعد كل جولة، وتحديث فوري للعدادات؛ البوتات مستثناة.
- الطاولات: 140 طاولة مع الحفاظ على المجموعة السابقة وإضافات v173.
- طربوش الباشا: 14 لوناً مستقلاً.
- المنافسات والمتجر: تذاكر المنافسات، الحزمة اليومية، الإعلانات والمصمم الشامل محفوظة ومفعلة ضمن بنية الإصدار.
