# Warqna V0.5 Professional

مشروع **ورقنا** الكامل للإصدار **0.5.0+500**: Flutter للويب وAndroid/iOS، وLaravel للخادم والاقتصاد والغرف والمنافسات.

## أبرز مكونات V0.5

- ألعاب الورق المعتمدة ومحركات خادمية للحركات والتوزيع.
- متجر مصنف مع شراء خادمي ورسائل رصيد دقيقة.
- طربوش الباشا الأحمر الأصلي وصناديق الجوائز اليومية.
- غرف مفتوحة وصور اللاعبين وألعاب رئيسية مفضلة.
- مجموعات بصور وصلاحيات متعددة وسجل نشاط.
- طريق تحدٍّ ومكافآت مستويات وتذاكر ومسرعات.
- لوحة Adnan والمصمم الشامل بدون كود وإدارة مفوضة.
- إعدادات اللغة والثيم والخط في جميع الصفحات.
- GitHub Actions لبناء Web وAPK وAAB وفحص Laravel والإصدار.
