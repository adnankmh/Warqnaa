# Warqna V175 Professional

نسخة موحدة مبنية على V174 وتضيف جدول XP الرسمي من Excel، توجيه دعوات الغرف مباشرة، تثبيت الاتجاه، تسجيل دخول ويب محلي تلقائي عند تعذر الخادم، حزم وتحديات خادمية، صورة الباشا الحمراء الكاملة، وإدارة شاملة للمدير.

ابدأ من `START_HERE_AR.md`، وشغّل فحوص Laravel وFlutter من GitHub Actions قبل النشر الإنتاجي.
