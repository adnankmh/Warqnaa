# Warqna V0.2 Professional

نسخة كاملة من لعبة **ورقنا** تجمع الميزات السابقة مع نظام **صندوق الجوائز اليومي** الجديد.

أبرز الإضافات:

- صفحة مستقلة لصناديق الجوائز من الصفحة الرئيسية، وليست داخل المتجر.
- صندوق واحد عن كل مباراة مكتملة يفوز بها اللاعب، بحد أقصى 4 صناديق يومياً.
- ستة صناديق ملكية متنوعة مع فتح أمامي سينمائي مدته 5 ثوانٍ.
- مكافآت: يوم باشا، لون كتابة، لون لاعب، غلاف شخصي، 50–1000 توكن بمضاعفات 50، أو تذكرة مسابقة 200.
- صور تذاكر فاخرة لجميع الفئات مع القيمة في المنتصف.
- طربوش الباشا الأحمر الأصلي فقط في واجهة المتجر، دون تلوين أو تبويب ألوان.
- ترجمة صفحة الصناديق ومكافآتها إلى العربية والإنجليزية والألمانية والتركية والفرنسية والإسبانية.
- اعتماد خادمي للصناديق والمكافآت مع منع التكرار وحد يومي وصلاحيات زمنية.
- المصمم الشامل للمدير يشمل صناديق الجوائز والتذاكر وباقي مكونات التطبيق.
