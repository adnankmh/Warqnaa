# Warqna V0.2.5 — Full Source

الإصدار الكامل من منصة ألعاب الورق الاجتماعية **ورقنا**، مع دمج الأنظمة السابقة وإضافة عدالة توزيع محسنة، رحلة تحديات متعددة المراحل، مكافآت مستويات، متجر ومعاينات مطورة، أدوات مظهر ثابتة، وإدارة محمية لحساب Adnan.

## المكونات

- `backend-laravel/`: الحسابات، الغرف، الألعاب، الاقتصاد، التحديات، المكافآت والإدارة.
- `flutter_app/`: تطبيق Flutter للويب وAndroid وiOS.
- `tools/`: فحوص الإصدار وعقود التوافق.
- `docs/`: أدلة التشغيل والإصدار والتحقق والحسابات التجريبية.
