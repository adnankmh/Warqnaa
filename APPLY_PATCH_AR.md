# طريقة تركيب تحديث V0.2.1

1. افتح مجلد مشروع **Warqna V0.2** الأساسي.
2. فك ضغط ملف التحديث في جذر المشروع نفسه.
3. وافق على **استبدال الملفات الموجودة** مع الحفاظ على المسارات كما هي.
4. نفّذ أو شغّل عبر GitHub Actions:

```bash
cd flutter_app
flutter pub get
flutter analyze

cd ../backend-laravel
php artisan migrate --force
php artisan test
```

## ملاحظة مهمة

ملف التحديث لا يحتوي على المشروع الكامل. لا تحذف ملفات المشروع القديمة؛ فقط انسخ محتويات التحديث فوق نسخة V0.2.
