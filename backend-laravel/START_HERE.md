# Warqna Laravel Platform v58 Ready

## Windows
1. فك الضغط.
2. افتح المجلد `warqna-laravel-platform-v58-pro`.
3. شغّل `setup-windows.bat` مرة واحدة فقط.
4. شغّل `start-windows.bat` كل مرة تريد تشغيل المنصة.
5. افتح: http://127.0.0.1:8000
