## التشغيل على Windows/XAMPP
1. فك الضغط داخل:
   `C:\xampp\htdocs\warqna-zone-pro-playable-v17`
2. شغّل Apache من XAMPP.
3. افتح PowerShell:
   ```powershell
   cd C:\xampp\htdocs\warqna-zone-pro-playable-v17\server
   npm install
   npm run dev
   ```
4. افتح الموقع:
   `http://localhost/warqna-zone-pro-playable-v17/public/index.php`

## ما الجديد في v17
- صفحة ألعاب وغرف بشكل احترافي قريب من منصات الألعاب مع تصميم أصلي.
- تنظيف الألعاب غير المناسبة: لا توجد ألعاب غير ورقية تعمل بورق الشدة مثل الشطرنج.
- 20 لعبة في الواجهة، وكل لعبة لها عدد لاعبين وطاولة/غرفة مناسبة قدر الإمكان.
- تحسين أماكن أزرار الطلب في الطرنيب حتى لا تغطي الورق.
- تكبير الورق المرمي وتحسين قراءة الطاولة.
- بروفايل منبثق داخل نفس الصفحة مع إحصائيات ألعاب، XP، المستوى، والتوكنز.
- حساب الإدارة يحتوي 100 مليار توكنز ومستوى 90 وإحصائيات ألعاب كبيرة.
- إرسال توكنز بين اللاعبين مع خصم 10% للإدارة.
- رسوم مسابقات 10% للإدارة.
- أصدقاء/قبول/رفض/حظر، وحظر اللاعبين من التواجد على نفس الطاولة في منطق قاعدة البيانات.
- ترجمة موسعة للواجهات الأساسية للغات: عربي، إنجليزي، فرنسي، تركي، إسباني، ألماني.

## ملاحظة قانونية مهمة
