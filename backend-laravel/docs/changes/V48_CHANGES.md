# Warqna v48 Pro

## أهم التعديلات
- إضافة خاصية مغادرة النادي للاعب مع حماية نادي المالك من الحذف الخاطئ.
- إضافة صفحة بحث عن اللاعبين بالاسم أو جزء من الاسم: `/players/search`.
- إضافة بحث سريع من الشريط العلوي.
- تحسين شكل اللاعبين والبوتات داخل الطاولة مع دائرة لونية فخمة حول الصورة والاسم حسب اللون المشترى.
- تفعيل ألوان الأسماء بصريًا حول اسم اللاعب داخل الغرفة ونتائج البحث.
- إعادة تنظيم طاولات المتجر إلى مستويات: مبتدئ، متوسط، محترف، أسطوري.
- إضافة ثيمات طاولات جديدة تعمل مباشرة عند تفعيلها من خانة مشترياتي.
- تحسين شكل الورق، الطاولة، الدردشة، أزرار اللعب، وسجل الحركات.
- تحديث نصوص قوانين الطرنيب بما يطابق منطق المحرك الحالي: طلب 7-13، Pass، اتباع النوع، الطرنيب، واحتساب اللمات.

## ملاحظات تقنية
- قاعدة SQLite المرفقة تم تحديثها بعناصر المتجر الجديدة.
- تم الحفاظ على سكربتات التشغيل السابقة.
- قوانين الطرنيب محسنة ومفعّلة في المحرك، أما الألعاب الأخرى فهي موضوعة ضمن محركات قابلة للتوسعة وتحتاج اختبار إنتاجي عميق قبل إطلاق عام.
