# v130 — Clean Luxury Rebuild

## لماذا هذه النسخة مختلفة
النسخ السابقة تراكمت فيها طبقات CSS وJavaScript متعددة، وبعضها كان يتعارض مع بعض. في v130 تم تنظيف أهم التعارضات وإعادة بناء نقاط الواجهة الأساسية بطبقة واضحة وحاسمة.

## 1. قائمة الألعاب
- تم حذف بقايا الكود القديم الذي كان يظهر بعد قائمة الألعاب ويسبب طفحًا.
- يوجد الآن عنصر واحد فقط لقائمة الألعاب: gamesCurtain.
- القائمة تظهر كنافذة مؤقتة فقط عند الضغط على زر الألعاب.
- تختفي عند:
  - اختيار لعبة.
  - الضغط خارجها.
  - زر Esc.
- البحث والتصنيفات داخل القائمة تعمل من جديد بأسماء v130 نظيفة.

## 2. صفحة الألعاب
- أزيلت واجهة sidebar القديمة.
- تم إنشاء واجهة جديدة:
  - wz-lobby-v130
  - wz-games-wall-v130
- الصفحة داخل الشاشة بدون طفح.
- فيها:
  - بحث.
  - تصنيفات.
  - وضع ضغط البطاقات.
  - فلتر الألعاب المميزة.
  - بطاقات ألعاب فخمة ومضغوطة.

## 3. التصميم العام
- NAV BAR في الأعلى كما طلبت.
- تم تقليل زحمة الروابط.
- منع overflow-x على الصفحات الأساسية.
- تحسين شكل الخلفيات والبطاقات.

## 4. الثيمات والدردشة
- تم تثبيت سلوك الثيمات من جديد:
  - اختيار الثيم يتفعل مباشرة.
  - يحفظ في localStorage ويرسل preference.
- الدردشة:
  - إغلاق أو تصغير يرجعها لأيقونة.
  - الضغط على الأيقونة يعيد فتحها.

## 5. محركات الألعاب
- v129 أضاف محركات أقوى للدومينو، اللمات، الرقعة، وألعاب أونو/دييل.
- v130 أضاف اختبارات للتأكد أن كل ألعاب GameCatalog تبدأ بمحرك وحالة turn صالحة.

## 6. اختبارات
- V130CleanLuxuryRebuildTest
- يفحص:
  - عدم وجود أكثر من قائمة ألعاب.
  - عدم وجود بقايا curtain قديمة.
  - صفحة الألعاب الجديدة.
  - تشغيل كل محركات الألعاب بحالة بداية صالحة.

## ملاحظة مهمة
هذه نسخة تنظيف وإعادة بناء واجهة قوية. ما زال الوصول إلى دقة 100% لكل لعبة مثل تطبيقات عالمية يحتاج اختبار لعب يدوي مطول لعبة لعبة، لكن البنية الآن صارت أنظف وأثبت من النسخ السابقة.
