# تحديث v114 — تحسين الدردشة والأصدقاء واللعب والقوانين

## مهم
تم تحسين تجربة الموقع لتكون أقرب لتجربة منصات ألعاب الورق الاجتماعية الاحترافية، مع الحفاظ على اسم Warqna وشكله الخاص وعدم نسخ أي شعار أو اسم أو مواد مملوكة لمنصة أخرى.

## ما تم تنفيذه
1. تحسين مركز الدردشة:
   - شكل اجتماعي أكثر احترافية.
   - رسائل اللاعب تظهر بوضوح ولون الكتابة الخاص به.
   - رأس الدردشة قابل للسحب.
   - الدردشة قابلة للتكبير والتصغير اليدوي.
   - عند إغلاقها تظهر أيقونة لإعادتها.
   - إضافة مؤشرات اجتماعية مثل "متصل".
   - إضافة فلترة كلمات مسيئة أوسع في الدردشة الخاصة ودردشة الغرف.

2. تحسين نظام الأصدقاء:
   - طلب صداقة / قبول / رفض / إلغاء / حظر / إلغاء حظر.
   - الأزرار تتغير بعد تنفيذ الإجراء.
   - تعامل AJAX قدر الإمكان بدون تحديث الصفحة.
   - شكل أوضح وأقرب للأنظمة الاجتماعية.

3. تحسين محرك الطرنيب:
   - إعادة كتابة محرك الطرنيب بشكل أكثر وضوحًا وثباتًا.
   - الطلب من 7 إلى 13 فقط.
   - الذي يمرر لا يعود للطلب في نفس الجولة.
   - صاحب أعلى طلب يختار الطرنيب.
   - اتباع نوع أول ورقة إلزامي إذا كان موجودًا في يد اللاعب.
   - الطرنيب يتفوق على باقي الأنواع.
   - الفائز باللمة يبدأ اللمة التالية.
   - احتساب نجاح/فشل الطلب والفريق الفائز.
   - الانتقال التلقائي للجولة التالية حتى الهدف.
   - رسائل الخطأ أصبحت واضحة ومفهومة.

4. تحسين الهاند والبناكل:
   - ترتيب الورق حسب سنك، ديناري، بستوني، كبة.
   - رسائل خطأ أوضح.
   - ترتيب اليد بعد السحب.
   - توضيح منطق السحب والتنزيل والرمي.

5. تحسين بقية الألعاب:
   - توحيد ترتيب الورق في القاعدة العامة.
   - إضافة ملف إرشادي لقواعد الألعاب:
     `config/warqna_game_rules.php`
   - تحديث صفحة قوانين الألعاب لعرض قواعد مفصلة للألعاب الأساسية.

6. تحسين واجهة اللعب:
   - الورق أصبح أكثر وضوحًا واحترافية.
   - إبراز الأوراق القانونية.
   - تحسين شكل الطاولة بصريًا.
   - عدم إظهار أخطاء جافة للمستخدم قدر الإمكان، بل رسائل مفهومة.

7. تحسين الإشعارات والرسائل:
   - تحويل رسائل الدردشة الخاصة إلى صفحة الأصدقاء بدل البحث.
   - رسائل الغرفة تعرض بيانات اللاعب واللون والصورة بشكل أغنى.

## ملاحظة فنية
لا يمكن تنفيذ كل قوانين كل ألعاب العالم بنسبة 100% في خطوة واحدة، لكن تم الآن تحسين أهم محرك وهو الطرنيب، وتقوية القواعد الأساسية للهاند والبناكل وباقي ألعاب الورق، وتجهيز ملف قواعد مركزي لتطوير كل لعبة بشكل مستقل في النسخ القادمة.
