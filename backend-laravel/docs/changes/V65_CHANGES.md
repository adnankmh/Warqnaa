# Warqna v65 Pro – Rules & Realistic Tables Update

## أهم التعديلات

- مراجعة أعمق لمحركات اللعب، خصوصًا الطرنيب والهاند والبناكل والبوتات.
- تحسين منطق البوتات: اختيار حركة قانونية بدل أول حركة عشوائية، ومحاولة اتباع اللون/اختيار حركة مناسبة.
- تحسين الحركة التلقائية عند تأخر اللاعب لتستخدم نفس منطق البوت القانوني قدر الإمكان.
- تحديث الهاند إلى 5 جولات مع احتساب -30 للفائز و -60 للهاند الكامل، والفائز النهائي هو الأقل نقاطًا.
- إلزام الهاند/البناكل بالسحب قبل الرمي، ومنع الرمي قبل السحب.
- حفظ نقاط الهاند/البناكل بين الجولات عند بدء الجولة التالية.
- إضافة طاولات واقعية وفخمة: خشب ماهوجني، جلد أسود، كازينو، مخمل أزرق، نقش عربي، رخام أسود.
- إضافة طاولات باسم اللاعب: ذهب كبير وفضة كبيرة.
- إضافة طاولات متحركة خفيفة: أسد، حصان، تنين.
- إضافة ظهور ورق واقعية: أحمر، أزرق، أسود ذهبي، فضي، توقيع ذهبي.
- تحسين شكل البوتات بصريًا داخل الطاولة.
- تحديث نصوص قوانين الهاند والبناكل في صفحة قوانين الألعاب.
- تحديث Seeder وقاعدة SQLite الجاهزة بعناصر v65 الجديدة.

## ملاحظة

القوانين مطبقة داخل Warqna بصياغة مستقلة ومنطق خاص بالمشروع، مع الاعتماد على القواعد العربية الشائعة للألعاب دون نسخ كود أو أصول من أي منصة أخرى.
