# تقرير تحديث Warqna v92

تم تنفيذ حزمة تعديلات كبيرة بناءً على طلب الإدارة، مع الحفاظ على التشغيل على port 8000.

## الألعاب والقوانين
- تفعيل ترتيب وتبديل أوراق الهاند/البناكل بصريًا عبر السحب والإفلات وتحديد الأوراق.
- إضافة اعتماد مجموعات جاهزة للنزول على الأرض في الهاند/البناكل من خلال `arrange_melds`.
- توسيع المجموعات القانونية من 3 أوراق إلى مجموعات أكبر عند تكوين set/run صحيح.
- تحسين نقاط البناكل: 3-6 = 0.5، 7-A = 1، 2 = 2، Joker = 4.
- تحسين الطرنيب: من يعمل Pass في مرحلة الطلب لا يرجع يطلب في نفس الجولة.
- إخفاء أزرار طلب 0-6 في الطرنيب حتى لا تظهر رسالة طلب خاطئة.
- تحسين اختيار الطرنيب والرسائل المتعلقة بمرحلة الطلب والطرنيب.
- بدء اللعبة مباشرة من زر البداية بدون نافذة تأكيد.

## الغرف والجلوس والخروج
- منع نقل اللاعب تلقائيًا إلى مقعد آخر عند اختيار مقعد غير مناسب.
- اللاعب الحالي يبقى مرتبطًا بمقعده ولا يتم تغيير مقعده تلقائيًا.
- عند إغلاق الصفحة أو الانقطاع، تبقى اللعبة مستمرة والبوت يلعب بدل اللاعب عند دوره.
- بعد 3 لفات غياب، يتحول مقعد اللاعب إلى بوت مؤقتًا مع إمكانية عودته إلى نفس المقعد.
- حد العودة/الخروج لنفس الغرفة 3 مرات، وبعدها يتم منع العودة لنفس الغرفة.
- إلغاء معاينة الطاولة من الغرف/اللعبة.

## النقاط والبروفايل
- عند انتهاء الجولة تضاف نقاط XP وتوكنز صغيرة تلقائيًا للبروفايل.
- تظهر نافذة صغيرة جدًا لمدة ثانيتين للاعب توضح نقاط الجولة والتوكنز المكتسبة.

## الدردشة
- تحسين مرونة الدردشة وتصميمها وحركتها.
- تحسين شكل الرسائل والإيموجي والتكبير/التصغير.
- تحسين ألوان التبويبات وحركة صندوق الدردشة.

## المتجر ولوحة الإدارة
- إضافة باقة باشا 1 يوم بسعر 1700 توكنز.
- إضافة باقة باشا 3 أيام بسعر 5000 توكنز.
- إضافة معاينة أوضح في إدارة المتجر قبل الإضافة/التعديل.
- دعم رفع صورة للطاولة وصورة لظهر الورق من الإدارة، وتظهر في المتجر ولدى اللاعبين بعد التفعيل.
- إضافة مفاتيح تحكم إدارية إضافية: بدء مباشر، نقاط الجولة الصغيرة، رفع صور الطاولات، رفع صور الورق.
- تحسين عرض صور الطاولات وظهر الورق المرفوعة داخل المتجر والمشتريات.

## الثيمات والترجمات
- تقوية أزرار الثيمات واللغات، وربطها بالتخزين المحلي وتفضيلات المستخدم.
- تحسين ظهور نوع اللعبة والأزرار حتى لا تكون بيضاء وغير واضحة.

## التشغيل
- تم الرجوع إلى port 8000 في:
  - `.env`
  - `.env.example`
  - `start-windows.bat`
  - `start-linux-mac.sh`
  - `README.md`
  - `START_HERE.md`

## الفحص
- تم فحص PHP Syntax لكل ملفات `app`, `routes`, `database` بنجاح.
- تم فحص JavaScript الأساسي `public/assets/js/app.js` باستخدام `node --check` بنجاح.

## ملاحظة مهمة
هذه التعديلات تقرّب المنصة من تجربة مواقع الألعاب الاجتماعية الكبيرة، لكنها ليست مطابقة حرفيًا لأي منصة تجارية. القوانين تم تحسينها داخل المحرك الحالي للمشروع، وقد تحتاج لاحقًا لاختبار لعب مباشر طويل لكل لعبة لضبط الحالات النادرة جدًا.
