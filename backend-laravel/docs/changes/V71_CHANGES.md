# Warqna v71 Pro

- تنظيم المسرعات إلى 6 فقط بصيغة xN: x1.25 / x1.5 / x2 / x3 / x4 / x5.
- مدة تشغيل المسرع 24 ساعة وصلاحيته في البيانات 10 أيام.
- إضافة تبويبات إيموجي: مبتدئ، متوسط، محترف، أسطوري، متحرك، كبير.
- تحسين معاينة المتجر لتظهر كمعاينة على بروفايل اللاعب.
- تصغير نافذة البروفايل وإظهارها في منتصف الشاشة وإغلاقها بالضغط خارجها أو Esc.
- تحسين مقاعد اللاعبين والبوتات بحيث لا تغطي الطاولة.
- تحسين الورق: صفان للهاند/البناكل/كونكان وصف واحد لباقي الألعاب.
- تحسين خانات المجموعات في الهاند والبناكل بإطار مميز.
- إضافة وضع الغائب للباشا: الكمبيوتر يلعب بدل اللاعب مؤقتًا.
- تحسين رسائل الأخطاء الودية عند عدم كفاية التوكنز أو الحركة غير القانونية.
- إضافة طاولات واقعية كبيرة للنسر والنمر والفهد والزجاج الذهبي.
- تحسين واجهة النوادي والمسابقات بصريًا.
- فلترة أولية للكلمات المسيئة داخل الدردشة.
