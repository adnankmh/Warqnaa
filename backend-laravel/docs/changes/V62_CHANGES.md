- إضافة 10 شارات إضافية فاخرة فوق شارات v60.
- إضافة 10 مؤثرات فوز إضافية فخمة ومتحركة.
- إضافة طاولات أسطورية جديدة بحيوانات ورموز داخلية: نمر، صقر، حوت، عقرب، نمر أسود، طاووس.

- تحسين اختيار البوت للورقة القانونية بدل اللعب العشوائي بأول ورقة.
- تحسين حركة اللاعب التلقائية عند التأخر حتى تختار حركة قانونية قدر الإمكان.
- تحسين مظهر البوتات داخل الطاولة بتأثير 3D.

- تعزيز CSS للشارات والمؤثرات والطاولات الأسطورية بدون ملفات خارجية أو أصول مملوكة.
