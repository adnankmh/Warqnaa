# تحديث v124 — Atlas Luxury World Platform

## لماذا هذه النسخة مختلفة عن v123
في v124 لم يتم الاكتفاء بتعديل ألوان أو CSS. تمت إضافة طبقات برمجية وفحص وتشغيل جديدة:

## 1. تصميم عالمي جديد
- طبقة Atlas Luxury كاملة على كل الموقع.
- إعادة شكل الهيدر والـ userbar.
- ضغط واجهة الألعاب والمتجر والغرف لتقليل التداخل.
- ألوان مختلفة بوضوح: خلفية داكنة عالمية + زمردي + ذهبي.
- لا horizontal overflow.

## 2. صالة الألعاب
- تشغيل ultra dense افتراضيًا.
- البطاقات أصغر وأكثر احترافية.
- كل الألعاب تظهر داخل نفس الإطار.
- مناسبة للشاشة والـ APK.

## 3. المتجر
- بطاقات أصغر وأفخم.
- شبكة مضغوطة.
- معاينات قوية بقيت موجودة.
- لا خروج من الإطار.

## 4. غرفة اللعب
- تحويل الغرفة إلى Arena:
  - معلومات يسار.
  - طاولة وسط.
  - دردشة يمين.
  - اليد وأزرار اللعب مثبتة داخل الطاولة.
- تحسين العرض بدون overflow.
- تصميم طاولة فخم.

## 5. الدردشة
- إغلاق نهائي كامل.
- إعادة فتح من أيقونة فقط.
- تثبيت السلوك حتى لو كانت سكربتات قديمة تحاول فتحها.

## 6. المحركات
- إضافة PlayActionNormalizer.
- كل حركة تمر على normalizer قبل validate:
  - bid:9 تتحول إلى bid + value.
  - choose_trump يقبل suit/trump/type.
  - play_card يقبل card/card_id/id/rank+suit.
  - play_tile يقبل tile/domino/id.
- تقليل أخطاء الورقة الصحيحة بسبب اختلاف تنسيق الواجهة.

## 7. فحص محركات داخل الإدارة
- إضافة:
  - EngineAuditController
  - /admin/engine-audit-v124
- يفحص كل الألعاب ويعرض هل المحرك يرجع initial state صالح.

## 8. اختبارات
- V124NormalizerAndAuditTest
- اختبار normalizer للطرنيب والطلبات.

## فحص البناء
- PHP Syntax OK.
- JavaScript Syntax OK.
- PWA/APK readiness OK.
