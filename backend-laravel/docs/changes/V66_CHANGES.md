# Warqna v66 Pro – Tarneeb Gameplay Hotfix

## أهم الإصلاحات
- إصلاح قبول الطلبات في الطرنيب وتقليل ظهور رسالة "الحركة غير مسموحة الآن" بدون سبب واضح.
- إذا كان الدور على بوت أثناء الطلب أو اللعب، يتم تشغيل البوت تلقائيًا حتى يصل الدور للاعب الحقيقي أو المرحلة التالية.
- عند بدء اللعبة بدون تحديث الصفحة، تظهر البوتات فورًا في المقاعد داخل نفس الصفحة.
- تحسين أشكال البوتات باستخدام صور Bot شبه واقعية بدل الشكل الافتراضي الواحد.
- عرض الورقة المرمية أمام كل لاعب في مكانه على الطاولة.
- عرض آخر لفة/جولة سابقة بشكل صغير على جانب الطاولة بدون تغطية اللعب.
- تحسين رسائل الخطأ: ليس دورك، اختر الطرنيب، الطلب يجب أن يكون أعلى، أو يجب اتباع النوع.
- البروفايل داخل نفس الصفحة يعرض التفعيلات والمقتنيات الحالية بشكل أوضح.
- تحسين تنسيق بطاقات الورق المرمي والبوتات والبروفايل المصغر.

## ملاحظات
هذه النسخة مخصصة لإصلاح تجربة الطرنيب واللعب اللحظي، وستبنى عليها نسخة لاحقة لمراجعة محركات الألعاب الأخرى لعبة لعبة.
