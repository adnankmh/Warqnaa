# Warqna v64 Pro

- إضافة تعديل كامل للبروفايل: الدولة، اللعبة المفضلة، الإيميل، كلمة المرور، وصورة اللاعب.
- إضافة عمود favorite_game_key للبروفايل مع Migration وتحديث قاعدة SQLite الجاهزة.
- جعل معاينة الطاولات تظهر بحجم الطاولة الرسمية داخل الغرفة مع خيارات معاينة متعددة.
- إغلاق معاينة الطاولة بالضغط خارجها أو زر Esc.
- إضافة طاولات باسم اللاعب: ذهبية/فضية/ألوان مدمجة.
- إضافة تبويبات فرعية للطاولات حسب المستوى، وتبويبات للإيموجي حسب النوع.
- تحديث المسرّعات: +25%, +50%, +100%, +150%, +200%, +300%, +400%, +500%، مدة التفعيل يوم واحد وصلاحية 10 أيام.
- إضافة أيقونات علوية للإشعارات/الدعوات/المسابقات/الرسائل داخل نفس الصفحة.
- إزالة منطقة مجموعات الهاند من غرف الطرنيب.
- تحسين عرض الإيموجي، البروفايل، الطاولات، والواجهة العامة.
