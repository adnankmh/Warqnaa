# Warqna v57 Pro

## أهم التحديثات

- دمج لون إطار اللاعب مع لون اللاعب في عنصر واحد باسم **ألوان اللاعبين + Glow**.
- عند تفعيل لون لاعب جديد يتم تفعيل:
  - لون الاسم.
  - Glow مضيء متحرك حول صورة البروفايل.
  - Glow حول اسم اللاعب داخل الموقع والطاولة.
- إضافة ألوان Glow جديدة: Ruby, Crimson, Sapphire, Ocean, Emerald, Lime, Violet, Pink, Gold, Diamond, Fire, Galaxy.
- إضافة ألوان كتابة جديدة داخل الموقع والدردشة.
- إضافة طاولات جديدة: Obsidian, Neon City, Andalus, Oasis, Dragon, Diamond Elite.
- إضافة ظهور ورق جديدة: Obsidian, Neon, Arabic Pattern, Dragon, Diamond.
- إزالة رابط القوانين من الغرفة؛ قوانين الألعاب تظهر من الـ NAV BAR فقط في صفحة **قوانين الألعاب**.
- إضافة 5 ثيمات للموقع من الإعدادات:
  - Royal
  - Midnight
  - Emerald
  - Desert
  - Galaxy
- تحديث قاعدة SQLite الجاهزة بالعناصر الجديدة.
- إضافة migration جديد لدعم theme المستخدم.
- تحسين شكل الورق و Glow والثيمات بصريًا.
- فحص PHP و JavaScript بدون أخطاء syntax.

## ملاحظة
القواعد مكتوبة ومفعلة بصياغة أصلية داخل Warqna وفق القواعد العربية الشائعة للألعاب، بدون نسخ كود أو أصول أو تصميمات من منصات أخرى.
