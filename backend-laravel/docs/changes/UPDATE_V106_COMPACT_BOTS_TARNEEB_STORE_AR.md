# تحديث v106 — حذف صور البوت الثقيلة، ضبط المقاعد، الأزرار، المتجر، والطرنيب

## ما تم تنفيذه
- حذف صور البوت الثقيلة PNG/JPG/WebP التي كانت تكبّر حجم الملف.
- إنشاء صور بوت SVG خفيفة جدًا بدل الصور الكبيرة، بنفس الأسماء:
  معتصم، يمان، عدنان، عاصم، كنان، جميل، همام، معاذ، مصطفى، مهند.
- تحديث مراجع صور البوت في:
  - `app/helpers.php`
  - `app/Http/Controllers/RoomController.php`
- ضبط صورة البوت/اللاعب لتظهر داخل المقعد/الكرسي بدل الخروج منه.
- توحيد محاذاة النصوص والأيقونات داخل جميع الأزرار في المنتصف.
- تحسين تبويبات المتجر والمشتريات والمعاينات حتى تظهر داخل نفس الصفحة بدون تمدد مزعج.
- تحسين عرض الإشعارات كلوحة داخل الصفحة بدون سحب الصفحة للأسفل.
- تحسين إدارة المتجر والمعاينات داخل الإدارة.
- في الطرنيب:
  - بعد انتهاء الطلب يظهر اختيار نوع الطرنيب.
  - اختيار الطرنيب لا يقبل إلا من صاحب أعلى طلب.
  - اعتماد نوع الطرنيب ينقل اللعبة إلى مرحلة اللعب.
  - إخفاء أزرار الألعاب الأخرى من لوحة الطرنيب.
  - الحفاظ على ترتيب الورق: سنك/شجرة، ديناري، بستوني، كبة — ومن الأكبر للأصغر.
- نقل تقارير الإصدارات السابقة إلى:
  `docs/changes`

## ملاحظة
تم تقليل الحجم بحذف صور البوت الكبيرة، واستبدالها بملفات SVG خفيفة.
