# Warqna v79 Stabilized Pro

- إعادة كتابة ملف app.js بشكل نظيف لإصلاح أخطاء JavaScript التي كانت تعطل الدردشة والمتجر والأزرار.
- تفعيل تبويبات المتجر ومشترياتي والتفعيل بالـ Ajax دون مغادرة التبويب.
- تفعيل دردشة اللعبة ودردشة الأصدقاء والبحث داخل نفس نافذة الدردشة مع التصغير والتكبير والإغلاق وإعادة الفتح.
- إصلاح بروفايل اللاعب المنبثق ليفتح داخل نفس الصفحة ويغلق بالضغط خارج النافذة أو Esc.
- دعم عرض أعلام الدول من ملفات SVG الحقيقية وحذف إسرائيل من القوائم والتحقق.
- تحسين اختيار الدولة في التسجيل وتعديل البروفايل بعرض اسم الدولة والعلم وحفظه مباشرة.
- تحسين ستارة كل الألعاب داخل نفس الصفحة.
- تحسين رسائل الأخطاء حتى لا يتوقف الموقع عند حركة غير صحيحة أو نقص توكنز.
- تحسين عرض الورق: الهاند/البناكل/كونكان صفين، وباقي الألعاب صف واحد.
- تحسين مربعات مجموعات الهاند والبناكل بإطارات مميزة.
- تحسين مقاعد اللاعبين والبوتات حتى لا تغطي الطاولة.
- إصلاح معاينة المتجر لتظهر على نموذج بروفايل اللاعب.
- إضافة CSS Stabilized لتثبيت الثيمات والطاولات والدردشة والبروفايل.
- فحص JavaScript بواسطة node --check وفحص ملفات PHP بواسطة php -l.
