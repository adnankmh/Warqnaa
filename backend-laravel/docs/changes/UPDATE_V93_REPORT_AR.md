# تقرير تحديث Warqna v93

تم تنفيذ تحديثات v93 على النسخة السابقة مع الالتزام بأن يعمل المشروع على البورت 8000، وأن تكون ملفات الموقع كلها داخل مجلد واحد داخل الملف المضغوط.

## أهم التعديلات

1. الدردشة
- إخفاء الدردشة نهائياً من صفحة المتجر.
- إظهار دردشة الغرفة/اللعبة داخل صفحة اللعبة بشكل واضح ومفتوح تلقائياً للاعبين الموجودين في نفس الغرفة.
- إضافة تحميل آخر رسائل الغرفة من السيرفر عند المزامنة، وليس فقط الاعتماد على Socket.IO.
- إبقاء الدردشة مرنة وقابلة للتكبير والتصغير والتحريك.

2. معاينات المتجر والإدارة
- إرجاع معاينة الطاولات وظهر الورق وألوان الكتابة بعد أن كانت مخفية بسبب إخفاء معاينة الطاولة في صفحة اللعب.
- حصر إخفاء معاينة الطاولة داخل الغرفة فقط، مع بقاء معاينات المتجر ولوحة الإدارة فعالة.
- تحسين معاينة إدارة المتجر بشكل أوضح وأفخم.
- دعم معاينة فورية عند رفع صورة جديدة للطاولة أو ظهر الورق داخل الإدارة.

3. الثيمات واللغة
- نقل أيقونة الثيمات وأيقونة اللغة إلى أعلى يسار الشاشة بشكل ثابت وواضح.
- منع ازدحامها مع باقي عناصر الشريط.
- تحسين موضع لوحة الثيمات واللغة.

4. البروفايل
- تكبير نافذة بروفايل اللاعب.
- عند ضغط اللاعب على بروفايله تظهر تفاصيل أكثر: XP، الخسائر، أيام الباشا، الرصيد، والمقتنيات.
- عند الضغط على لاعب آخر تظهر نافذة أصغر ومناسبة.
- تعطيل فتح بروفايل البوت حتى لا يظهر كلاعب حقيقي.

5. إنشاء اللعبة
- عند الضغط على زر إنشاء اللعبة الكبير في منتصف الصفحة تظهر نافذة صغيرة فيها نفس خيارات إنشاء اللعبة الموجودة في الجانب.
- إنشاء الغرفة يتم من نفس النافذة مباشرة.

6. المسابقات
- تحسين أزرار المسابقات ومنع دخولها فوق بعضها.
- تكبير خانة تسجيل المسابقة وجعلها في منتصف البطاقة وأكثر وضوحاً.
- إضافة وصف واضح لنظام سجل المسابقة وإعادة المشاهدة النصية بعد انتهاء المسابقة.
- تجهيز سجل المسابقة لحفظ الحركات والنتائج ولقطات نهائية من أوراق اللاعبين بعد انتهاء مباراة المسابقة.

7. الألعاب والقوانين
- إبقاء تحسينات الطرنيب السابقة: الطلب من 7 إلى 13، والـ Pass يخرج اللاعب من طلب نفس الجولة.
- إظهار أزرار الطلب المناسبة حسب نوع اللعبة.
- استمرار إخفاء معاينة الطاولة داخل الغرف حسب طلبك السابق.
- عدم إظهار زر الجلوس مكان البوت للاعب الموجود أصلاً داخل الغرفة بعد تحديث المقاعد بالمزامنة.

8. Responsive
- تحسين استجابة الموقع للموبايل والتابلت.
- تحسين صفحات الغرف، المتجر، الإدارة، المسابقات، الدردشة، والبروفايل للشاشات الصغيرة.

## فحوصات تم تنفيذها

- فحص Syntax لكل ملفات PHP بنجاح.
- فحص JavaScript الرئيسي باستخدام node --check بنجاح.
- التأكد من بقاء التشغيل على: http://127.0.0.1:8000

## ملاحظة مهمة

تسجيل المسابقة كفيديو حقيقي يحتاج نظام تسجيل Canvas/Video أو توليد Replay مرئي من أحداث اللعبة. في هذه النسخة تم تجهيز سجل إعادة مشاهدة نصي/حدثي داخل بنية المشروع، ويمكن تطويره لاحقاً إلى Replay مرئي كامل أو فيديو MP4 إذا تم إضافة طبقة Recording مخصصة.
