# Warqna v67 Pro – Chat + Turn Sync Hotfix

## التحديثات
- تفعيل دردشة اللعبة داخل الغرفة مع حفظ الرسائل في قاعدة البيانات وإرسالها عبر Socket.IO عند تشغيل السيرفر.
- تفعيل دردشة الأصدقاء عالميًا من أي صفحة، مع قائمة أصدقاء، بحث عن لاعب، فتح محادثة، وإرسال رسائل خاصة للأصدقاء فقط.
- جعل نافذة الدردشة قابلة للتصغير والإغلاق المؤقت والعودة كأيقونة صغيرة أسفل الشاشة.
- إضافة تبويبات داخل الدردشة: دردشة اللعبة، الأصدقاء، البحث.
- إضافة مزامنة تلقائية للغرفة حتى لا تبقى عالقة على رسالة "بانتظار دور لاعب آخر" عندما يكون الدور على بوت.
- إضافة زر تحديث الدور داخل رسالة الانتظار.
- إضافة endpoint جديد لمزامنة حالة الغرفة وإكمال حركات البوتات.
- تحسين شكل قائمة الأصدقاء والبحث والمحادثات.
- الحفاظ على بروفايل اللاعب داخل نفس الصفحة بدون انتقال.

## ملاحظات تشغيل
- دردشة اللعبة اللحظية تحتاج تشغيل Socket.IO بالأمر: npm run socket
- بدون Socket.IO ستظل الرسالة تظهر لك محليًا ويتم حفظها في Laravel، لكن البث اللحظي الكامل يحتاج السيرفر.
