# Warqna Laravel v51 Pro

## أهم تحديثات v51
- فصل قوانين الألعاب عن صفحة الغرفة في صفحة مستقلة: `/game-rules`.
- تحويل شكل طاولة اللعب إلى طاولة مربعة احترافية بدل الشكل الدائري.
- تحسين شكل الورق 3D مع ظل وحركة Hover وتداخل طبيعي للأوراق.
- إبقاء أزرار طلب الطرنيب العملية من `Pass / 7` إلى `13`، مع استمرار إخفاء أزرار 0-6 إلا في ألعاب الطلب المناسبة مثل الاستيميشن.
- إضافة Auto-play بعد 7 ثوانٍ إذا تأخر اللاعب في دوره.
- تسجيل 3 تأخيرات على اللاعب وتحويله إلى منقطع مؤقتًا، ويمكنه العودة لنفس مكانه إذا لم يتم استبداله.
- إضافة خيار مغادرة الغرفة، وإلغاء الغرفة تلقائيًا إذا خرج كل اللاعبين الحقيقيين.
- إضافة خيار إنشاء لعبة صوتية في كل الألعاب مع خصم 200 توكنز من كل لاعب يدخلها.
- إضافة صفحة الإشعارات، الإعدادات، حول، واتصل بنا.
- تفعيل ألوان الاسم والكتابة بصورة أوضح في الشريط، البروفايل، الدردشة، وحول اسم اللاعب.
- تحسين البروفايل: الإحصائيات، المقتنيات، طلب صداقة، حظر، ورسالة خاصة للأصدقاء فقط.
- النوادي أصبحت بسعات واضحة حسب المستوى: 50، 100، 150، 200، 300 لاعب.
- تحسين تكبير الخط حتى 180% بدل الحد القديم الصغير.

## ملاحظة تشغيل
إذا كانت قاعدة البيانات موجودة من نسخة أقدم، شغّل:

```bash
php artisan migrate --seed
php artisan optimize:clear
```

أو استخدم ملف `reset-database-windows.bat` لإعادة إنشاء قاعدة البيانات بالكامل في بيئة ويندوز.
