# Warqna v63 Pro

- وضع زر توزيع الورق وبدء اللعبة في منتصف الطاولة، ويظهر مرة واحدة فقط في بداية اللعبة.
- بدء اللعبة عبر Ajax بدون تحديث الصفحة عند الضغط على زر البدء.
- جعل الورق واقفًا وغير مائل مع شكل أكبر وفخم.
- إضافة خيارات متعددة عند معاينة الطاولة، مع الإغلاق بالضغط خارج النافذة أو زر Esc.
- إصلاح ظهور أزرار الطلب والتمرير في الطرنيب بإضافة game_type لمحرك الطرنيب.
- تحديث قواعد الهاند: 5 جولات، الفائز -30، الهاند الكامل -60، مع دعم التبديل وتجميع الأوراق.
- تحسين عرض البروفايل داخل نفس الصفحة بنافذة مصغرة.
- تحسين عرض أعلام الدول بجانب اللاعبين.
- إضافة أيقونات علوية للإشعارات: نوادي، ألعاب، مسابقات، رسائل/أصدقاء.
- إضافة طاولات أسطورية إضافية ومعاينة لها.
- ضغط النسخة داخل مجلد واحد فقط داخل ملف zip.
