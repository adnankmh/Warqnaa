# Warqna v81 Admin Control Pro

- إضافة نظام ثيمات كامل للموقع: Royal, Midnight, Emerald, Desert, Galaxy, Crimson, Ocean, Obsidian, Aurora.
- إضافة تحكم إداري GUI بدون أكواد لإعدادات الموقع العامة.
- يمكن للمدير اختيار الثيم الافتراضي وفرضه على كل اللاعبين أو ترك كل لاعب يختار ثيمه.
- يمكن للمدير تشغيل/إيقاف المتجر، النوادي، المسابقات، الدردشة، والدعم من لوحة الإدارة.
- إضافة إدارة متجر GUI كاملة: إضافة، تعديل، إخفاء المقتنيات، تعديل الاسم، السعر، المدة، التبويب، المستوى، الأيقونة، الكلاس البصري، اللون، مضاعف XP، والإيموجي.
- يمكن التحكم بتبويبات المقتنيات مثل الطاولات والإيموجي والشارات من payload بدون تعديل ملفات.
- تحسين CSS للثيمات بحيث تؤثر على الخلفية، الشريط العلوي، الأزرار، البطاقات، والطاولات الافتراضية.
- إضافة جدول site_settings وموديل SiteSetting.
- تحديث Seeder حتى تعود إعدادات الموقع بعد reset.
