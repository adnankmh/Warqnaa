# Warqna v60 Pro – Visual Store & Chat Cosmetics Update

## التعديلات الرئيسية

- تكبير وتحسين مؤثرات الفوز وجعلها أكثر احترافية وفخامة.
- إضافة أكثر من 10 مؤثرات فوز داخل تبويب مؤثرات الفوز.
- إضافة أكثر من 10 شارات داخل تبويب الشارات.
- تحسين معاينة لون الكتابة داخل المتجر بحيث تظهر تحت أيقونة الكتابة/اليد وتوضح لون الكتابة المختار.
- تفعيل لون الكتابة داخل دردشة اللعبة بحيث يظهر لون رسالة اللاعب للجميع حسب اللون المفعل.
- تحسين تبويب إيموجي الدردشة داخل الدردشة ليظهر بنظام 4×5 واضح ومنظم.
- جعل دردشة اللعبة تظهر فقط داخل صفحة غرفة اللعبة، وليست في كل صفحات الموقع.
- تخفيف لمعان الـ Glow حول صورة اللاعب والبروفايل حتى يكون أجمل وأهدأ.
- تحسين أشكال الطاولات وإضافة طاولات جديدة باحترافية أعلى.
- إضافة طاولات برسومات/رموز داخلية مثل الأسد، الحصان، النسر، الذئب، العنقاء، التنين.
- إضافة طاولة أسطورية ذهبية مميزة باسم اللاعب بسعر مرتفع من التوكنز.
- تقسيم الطاولات بصريًا إلى مستويات: مبتدئ، متوسط، متقدم، احترافي، أسطوري.
- إضافة ظهور ورق جديدة: السلطان، موزاييك عربي، الوردة، النمر الأسود، السماء.
- تحديث قاعدة البيانات الجاهزة SQLite بالعناصر الجديدة.
- تحديث Seeder حتى تعود العناصر الجديدة بعد أي reset أو migrate:fresh --seed.

## ملاحظات

الأصول المرئية الجديدة مبنية بـ CSS و SVG/Emoji داخلية بدون ملفات خارجية أو أصول مملوكة.
