# تحديث v122 — نسخة فخمة مستقرة مستفيدة من السوق وملف جيميني

## المنطلق
تم البناء على v121 مع الحفاظ على ما كان مستقرًا في v115 للدردشة، ودمج تعليمات ملف جيميني، والاستفادة من أفكار تطبيقات ألعاب الورق الاجتماعية العربية بدون نسخ أسماء أو شعارات أو أكواد.

## التغييرات الأساسية
1. صفحة كل الألعاب
- إعادة تصميم كاملة.
- عرض كل الألعاب في نفس الصفحة.
- بحث سريع.
- تصنيفات أفقية واضحة.
- بطاقات ألعاب فخمة ومناسبة للموبايل.

2. كتالوج الألعاب
- توسيع الكتالوج إلى أكثر من 40 لعبة/تجربة.
- إضافة عائلات:
  - الطرنيب والشراكة.
  - ألعاب اللمّات.
  - الهاند والتجميع.
  - التريكس والعقود.
  - ألعاب الخليج.
  - ألعاب التقاط الورق.
  - الدومينو والرقعة.
  - ألعاب اجتماعية.
- ربط الألعاب الجديدة بمحركات آمنة في GameFactory.

3. الدردشة
- إغلاق كامل عند الضغط على زر الإغلاق.
- ظهور أيقونة واحدة فقط لإرجاع الدردشة.
- السحب والتصغير بقيت موجودة.
- هذا السلوك قريب من استقرار v115.

4. الطرنيب
- تقوية normalize للورق والطرنيب.
- قبول صيغ مثل:
  - clubs / c / ♣ / سنك
  - diamonds / d / ♦ / ديناري
  - spades / s / ♠ / بستوني
  - hearts / h / ♥ / كبة
- قبول card أو rank+suit أو card_id.
- تقليل خطأ رفض الورقة الصحيحة.

5. المتجر
- تحسين المعاينات:
  - طاولة كاملة.
  - ظهر ورق.
  - إيموجي.
  - مسرعات XP كمكوك 3D.
  - مؤثرات الفوز.
- تحسين استنتاج التصنيفات إذا لم تكن محددة صراحة.

6. APK/PWA
- الحفاظ على جاهزية v120:
  - Capacitor.
  - GitHub Actions.
  - PWA.
  - Mobile API.
