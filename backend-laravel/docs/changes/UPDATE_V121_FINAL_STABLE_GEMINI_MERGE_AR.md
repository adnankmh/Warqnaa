# تحديث v121 — النسخة النهائية المستقرة بدمج ملف تعليمات جيميني والرجوع لثبات v115

## لماذا هذه النسخة؟
تم بناء v121 على v120، مع إصلاح الأخطاء التي ظهرت، والرجوع لأسلوب v115 المستقر في الدردشة وتجربة الواجهة، ودمج محاور ملف تعليمات جيميني داخل Laravel الحالي.

## الإصلاحات الأساسية
1. صفحة كل الألعاب:
   - إعادة تصميم صفحة `/games`.
   - كل الألعاب تظهر داخل نفس إطار الصفحة.
   - تقسيم الألعاب إلى عائلات واضحة.
   - بحث وفلاتر بدون خروج أو تداخل.
   - عرض معلومات كل لعبة: اللاعبين، المحرك، الشراكة، الملخص.

2. الدردشة:
   - زر الإغلاق يخفي الدردشة بالكامل.
   - تظهر أيقونة صغيرة لإرجاعها.
   - التصغير والتحريك بقي بشكل مستقر مثل v115.
   - منع بقاء صندوق الدردشة ظاهرًا بعد الضغط على ×.

3. الطرنيب:
   - إعادة تقوية محرك `TarneebRules`.
   - إصلاح اختيار الطرنيب.
   - إصلاح مشكلة رفض الورقة الصحيحة.
   - إضافة normalization للورق والأنواع:
     clubs / ♣ / سنك
     diamonds / ♦ / ديناري
     spades / ♠ / بستوني
     hearts / ♥ / كبة
   - دعم أفضل لاتباع النوع وحساب الفائز باللفة.
   - رسائل خطأ أوضح.

4. المتجر:
   - إصلاح خطأ Blade في مشترياتي.
   - تحسين تصنيف الطاولات وظهر الورق حسب التبويبات.
   - تحسين تصنيف الإيموجي.
   - معاينات فخمة للطاولات، ظهر الورق، الإيموجي، والمسرعات.
   - المسرعات تظهر كمكوك 3D.
   - معاينة واقعية قبل الشراء.

5. الألعاب والمحركات:
   - توسيع كتالوج الألعاب:
     طرنيب، 41، 61، سوري، 400، هاند، هاند سعودي، بناكل، رومي، كونكان، تريكس، تريكس كمبلكس، بلوت، حكم، استيميشن، سبيدي، هارتس، ليخة، كوت، باصرة، دومينو، لودو، جاكارو، طاولة، شطرنج.
   - تحديث GameFactory لربط الألعاب بمحركاتها.
   - مزامنة تلقائية للألعاب من الكتالوج عند فتح صفحة الألعاب.

6. دمج ملف تعليمات جيميني:
   - تطبيق الهوية البصرية الفخمة.
   - التركيز على منصة اجتماعية للألعاب شبيهة بروح تطبيقات الورق الكبرى بدون نسخ الاسم أو الشعار.
   - إضافة ملف:
     `GEMINI_V121_MERGE_NOTES_AR.md`

هذه نسخة Laravel/PWA/APK-ready محسنة ومستقرة. هي لا تنسخ جواكر أو شعاره أو اسمه، لكنها تستفيد من فكرة منصات ألعاب الورق الاجتماعية: طاولات، غرف، متجر، أصدقاء، دردشة، قوانين، ومكافآت.
