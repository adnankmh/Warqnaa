# تحديث v105 — دمج ملفات التعليمات والمتجر والمنافسات والمجموعات

تم أخذ المتطلبات القابلة للتنفيذ من الملفات المرفوعة ودمجها في المشروع.

## المتجر
- إضافة ملف مرجعي `config/warqna_manifest.php` يحتوي على خريطة المتجر والمنافسات والمجموعات.
- تحديث الـ Seeder بعناصر متجر v105:
  - باقة باشا سنة 365 يوم.
  - 6 مسرعات XP فقط، مدة التفعيل يوم واحد وصلاحية 10 أيام.
  - ألوان اسم + Glow كمقتنى واحد.
  - ألوان كتابة دردشة جديدة.
  - طاولات جديدة موزعة على: مبتدئ، متوسط، مميز، احترافي، أسطوري، متحركة.
  - ظهور ورق جديد.
  - إيموجي مصنفة حسب: مجاني، ضحك، فرح، عصبية، حزن، VIP، متحرك.
- تحسين CSS/JS لفلاتر المتجر وإدارة المتجر حتى تعمل فعليًا.
- تحسين معاينات الطاولات وظهر الورق والألوان.

## المنافسات
- إضافة توزيع جوائز ونقاط صدارة داخل بيانات المنافسة.
- تحسين معاينة إنشاء المنافسة: يظهر عدد المقاعد المطلوب، رسوم الإنشاء، الجائزة المتوقعة، التوزيع التقريبي، ونقاط الصدارة.
- دعم مفهوم خروج المغلوب حتى 4 مراحل، مع مقاعد محسوبة تلقائيًا.

## المجموعات
- تعديل مستويات وسعة المجموعات حسب نظام دوري المجموعات:
  - برونزي، فضي، ذهبي، بلاتيني، ماسي، أسطوري.
  - سعة مبدئية 20 وترتفع حسب المستوى.
- إضافة حقول داعمة للمجموعة: league_tier، total_points، season_points، capacity، logo، description.
- تحسين صفحة المجموعة لإظهار نظام المجموعة، الخزنة، السعة، النقاط، و Bonus اللعب الجماعي.

## الإدارة
- دعم إعدادات إضافية: حجم الورق، شكل الإشعارات، تكبير البوتات، إظهار أزرار الطرنيب فقط.
- تحسين إدارة المتجر وفلاتر الطاولات والإيموجي.
- نقل تقارير التغييرات السابقة إلى `docs/changes` بدل المجلد الرئيسي.

## الواجهة
- تحسين نافذة الإشعارات لتظهر داخل نفس الصفحة بدون دفع الصفحة أو فتح صفحة جديدة.
- تحسين CSS للأزرار والتبويبات الكبيرة.
