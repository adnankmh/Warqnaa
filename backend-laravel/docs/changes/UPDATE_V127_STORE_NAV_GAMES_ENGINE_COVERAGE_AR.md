# تحديث v127 — فصل المتجر + إصلاح كل الألعاب في NAV BAR + فحص محركات

## 1. المتجر صار مفصولًا
تم إلغاء عرض كل المنتجات داخل صفحة واحدة.
الآن كل قسم يظهر لوحده:
- أيام الباشا وحدها.
- الطاولات وحدها.
- ظهر الورق وحده.
- ألوان الكتابة وحدها.
- ألوان الأسماء وحدها.
- الإيموجي وحده.
- الشارات وحدها.
- المؤثرات وحدها.
- مسرعات XP وحدها.
- مشترياتي وحدها.

كل قسم له شبكة منتجات خاصة، والبحث يعمل داخل القسم الحالي فقط.

## 2. قائمة كل الألعاب في NAV BAR
تم استبدال لوحة الألعاب القديمة بلوحة جديدة مضغوطة:
- تظهر كل الألعاب داخل لوحة واحدة.
- بحث سريع.
- تصنيفات.
- بطاقات صغيرة.
- لا يوجد طفح خارج الصفحة.
- يوجد زر لفتح صفحة الألعاب الكاملة.

## 3. محركات الألعاب
تم إضافة EngineCoverageService:
- يفحص كل الألعاب الموجودة في GameCatalog.
- يتأكد أن كل لعبة لها Engine يرجع initial state صالح.
- يعطي JSON من:
  /admin/engine-coverage-json-v127

## 4. اختبارات
تم إضافة:
- tests/Feature/V127StoreAndEngineCoverageTest.php

## 5. ما بقي مهم
المحركات العميقة لكل لعبة مثل البلوت والهاند والتريكس تحتاج اختبارات لعب تفصيلية طويلة لكل استثناء. v127 يضمن وجود محرك وفحص مبدئي لكل لعبة، ويمنع وجود لعبة بلا محرك أو بلا حالة بداية.
