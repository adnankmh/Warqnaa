# Warqna v85 Full UI Rules Admin

- توسيع نظام الترجمة الفورية في الواجهة وإضافة مفاتيح جديدة للأزرار الأساسية.
- تحسين الثيمات لتؤثر مباشرة على الدردشة، اللوحات العلوية، البطاقات، وواجهة الإدارة.
- إضافة حقول تحكم إضافية في لوحة الإدارة لهكيل الموقع: شكل التنقل، كثافة الواجهة، شكل البطاقات، اللغة الافتراضية، وشكل المتجر.
- تحسين بروفايل اللاعب المنبثق ليعرض العلم واسم الدولة والتقدم للمستوى والمقتنيات، مع نسخة أغنى لصاحب الحساب.
- تحسين أشكال البوتات بملفات SVG جديدة أكثر احترافية.
- توسيع وصف القوانين والترجمات في صفحة قوانين الألعاب، مع إبقاء القوانين خارج غرف اللعب.
- تحسين تبويبات الإيموجي داخل الدردشة وتفعيل الإيموجي الكبير والمؤثر الصوتي.
- إضافة تنسيقات CSS لتثبيت حجم البروفايل، تحسين الدردشة، تحسين ستارة الألعاب، ولوحات الثيم واللغة.
