# Warqna v56 Pro

- نافذة تأكيد قبل أي شراء، ورسالة نجاح بعد الشراء.
- إضافة باشا سنة كاملة 365 يوم بسعر 300,000 توكنز.
- إضافة معاينة قبل شراء أي عنصر.
- إضافة 40 إيموجي متنوع، منها مجاني ومنها مدفوع بأسعار 1,000 و5,000 و10,000 و15,000 توكنز، مع دعم الحركة في الواجهة.
- تفعيل ثيم الطاولة حسب أغلى طاولة مفعلة بين اللاعبين الموجودين في الغرفة.
- إضافة مؤقت الدور حسب سرعة الغرفة، مع دعم 5 ثوانٍ في الوضع السريع.
- تحسين ترتيب أوراق الطرنيب: سنك/شجرة، ديناري، بستوني، كبة، ومن الأكبر للأصغر داخل كل لون.
- إضافة اختيار سرعة اللعبة عند إنشاء الغرفة.
- تحسين لوحة الإدارة بتبويبات: اللاعبين، الغرف المفتوحة، النوادي، المسابقات، الحماية، مع إجراءات اللاعب.
- تحسين المتجر والتبويبات والأشكال والألوان.
