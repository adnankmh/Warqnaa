## 2. لوحة الإدارة Pro
تم تقوية صفحة الإدارة:
- تبويبات الإدارة تعمل.
- ظهور الغرف المفتوحة والشغالة.
- ظهور النوادي الحالية.
- ظهور المسابقات المفتوحة والشغالة.
- كل اللاعبين يظهرون ببطاقات أكبر.
- الضغط على صورة أو اسم اللاعب يفتح البروفايل.
- يمكن تعديل:
  - الاسم الظاهر.
  - المستوى.
  - XP.
  - لون الاسم.
- يمكن:
  - حظر / فك الحظر.
  - حذف نهائي.
  - إرسال توكنز مخفية من الإدارة.
  - إضافة صداقة مباشرة.
