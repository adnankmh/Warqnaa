# Warqna Laravel Platform v52 Pro

## Main updates
- Added a lightweight professional WebAudio sound system without external copyrighted files.
- Sounds include: card throw, draw, discard, bidding, win, lose, message, notification, shop purchase, and UI click.
- Added sound on/off controls from the top bar and Settings page.
- Added `sound_enabled` user preference in profile/migration/database.
- Improved store economy with higher cosmetic prices and new premium items.
- Added new store categories: card backs, name frames, victory effects, and emoji packs.
- Activating colors now keeps name color and chat/text color active through profile CSS variables.
- Added active card back and active effect support inside the game table.
- Added richer 3D card backs and premium CSS effects.
- Added multi-language rule summaries for Arabic, English, Turkish, French, German, and Spanish.
- Improved the rules page to show translations inside each game rule card.
- Added daily engagement cards on the games page to encourage repeat play.
- Updated the ready SQLite database with new columns and store items.

## Important note
Rules and flows are original Warqna implementations inspired by common regional card/board game rules. No proprietary code, assets, or protected design files from any third-party platform are copied.
