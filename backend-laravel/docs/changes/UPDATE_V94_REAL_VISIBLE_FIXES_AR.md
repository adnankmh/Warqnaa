# تحديث v94 — إصلاحات مرئية حقيقية

تم تنفيذ التعديلات مباشرة داخل ملفات المشروع وليس كتقرير فقط.

## أهم التعديلات
- الحفاظ على التشغيل على البورت 8000.
- إظهار دردشة خاصة داخل صفحة الغرفة نفسها لجميع لاعبي الغرفة.
- إخفاء الدردشة بالكامل من صفحة المتجر وإرجاعها في باقي الصفحات.
- نقل أزرار الثيمات واللغات إلى أعلى يسار الشاشة فقط وبشكل ثابت.
- تحسين معاينة المتجر: ظهر الورق، الطاولة، لون الكتابة، لون الاسم، والصور المرفوعة.
- إصلاح تصميم إدارة المتجر وصفحة إدارة الطاولات/ظهر الورق ومعاينتها.
- إضافة معاينة إدارية مباشرة عند تغيير اللون/الأيقونة/الصورة/الكلاس.
- تكبير وتحسين نافذة البروفايل، مع تمييز بروفايل اللاعب نفسه عن بروفايل اللاعبين الآخرين.
- إضافة نافذة إنشاء لعبة وسط الصفحة بنفس خيارات إنشاء اللعبة الموجودة في الجانب.
- تحسين أزرار المسابقات ومنع تداخلها.
- تكبير خانة سجل المسابقة وجعلها واضحة.
- إضافة صفحة تسجيل/إعادة مشاهدة للمسابقة مع توليد فيديو WebM من سجل الحركات داخل المتصفح.
- إضافة Route جديد: /tournaments/{tournament}/replay
- إضافة Canvas/MediaRecorder لتوليد فيديو إعادة المشاهدة.
- حفظ Frames للحركات داخل حالة غرفة المسابقة عند تفعيل التسجيل.
- تحسين Responsive للموبايل والتابلت لصفحات الغرف، المتجر، الإدارة، النوادي، المسابقات، والدردشة.
- إبقاء عناصر باشا 1 يوم و3 أيام في قاعدة البيانات.

## ملاحظات مهمة
- توليد الفيديو يتم داخل المتصفح بصيغة WebM من خلال MediaRecorder، وليس MP4 server-side. هذا مناسب للتجربة على XAMPP بدون FFmpeg.
- إذا أردت MP4 حقيقي على السيرفر لاحقًا، نحتاج إضافة FFmpeg و endpoint تخزين فيديوهات.
