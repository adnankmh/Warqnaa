# Warqna v49 Pro - Game Rules Upgrade

تم في هذه النسخة توسيع محركات القواعد لكل الألعاب بدل الاعتماد على محرك واحد للطرنيب فقط.

## المحركات المضافة/المحدثة
- TarneebRules: طلب 7-13، Pass، اختيار الطرنيب، اتباع النوع، حساب اللمات والنقاط.
- Tarneeb400Rules: هدف 400 مع نفس آلية الطرنيب العامة.
- Tarneeb41Rules: هدف 41.
- EstimationRules: طلب فردي لكل لاعب، اتباع النوع، حساب نقاط تحقق الطلب أو الفشل.
- TrixRules: عقود تركس: شيخ الكبة، البنات، الدنانير، الكبة، اللطوش، تركس.
- HeartsRules / Leekha: عقوبات القلوب وملكة البستوني.
- BalootRules: 32 ورقة، صن/حكم، اختيار الحكم، اتباع النوع، حساب لفات مبسط.
- HokmRules: اختيار الحكم ثم اللعب بالشراكة.
- KoutRules: طلب ولفات للفرق.
- BasraRules: لاعبان، أربع أوراق، طاولة، جمع نفس الرقم وحساب النقاط.
- HandRules / Konkan: سحب ورمي، نزول 51، مجموعات وسلاسل.
- PinochleRules / Banakil: محرك مبني على السحب والرمي والمجموعات مع هدف 222.
- SimpleTurnRules: محرك دور آمن للدومينو والطاولة والتدريب إلى حين إضافة واجهة لوحة كاملة.

## الواجهة
- أزرار الطاولة أصبحت تتغير حسب مرحلة اللعبة: مزايدة، اختيار طرنيب/حكم، اختيار عقد تركس، صن/حكم للبلوت، سحب/رمي للهاند.
- نص القواعد في الغرفة صار يأتي من قاعدة بيانات اللعبة بدل نص الطرنيب الثابت.

## ملاحظة
القواعد مصاغة ومبنية بشكل أصلي مستوحى من القواعد الشائعة لألعاب الورق العربية ومن صفحات القواعد العامة المتاحة، وليست نسخًا حرفيًا من أي منصة أخرى.
