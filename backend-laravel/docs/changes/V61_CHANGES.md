# Warqna v61 Pro

- إزالة حقل رسوم الدخول من إنشاء الغرفة وجعلها 0 افتراضياً.
- جعل أقل مستوى قائمة منسدلة من 1 حتى مستوى منشئ الغرفة، وبحد أقصى 100.
- في الطرنيب أصبحت نهاية اللعبة قائمة منسدلة: 31 / 41 / 61 فقط.
- تكبير عناصر وأزرار إنشاء الغرفة وتحسين الشكل العام.
- تثبيت عدد المقاعد الثابت للألعاب التي لا تسمح إلا بعدد محدد.
- تحسين شبكة إيموجي الدردشة 4×5 تقريباً وتخفيف وهج صورة اللاعب.
- تكبير معاينة مؤثرات الفوز والشارات داخل المتجر.
