# تحديث v103 — صور البوت الملكية واستكمال تحسينات الواجهة

## ما تم
- إضافة 10 صور بوت ملكية جديدة بأسماء عربية فعلية: معتصم، يمان، عدنان، عاصم، كنان، جميل، همام، معاذ، مصطفى، مهند.
- أي بوت جديد أو استبدال لاعب ببوت أو خروج لاعب وتحوله إلى بوت سيستخدم واحدة من هذه الشخصيات وأسمائها.
- تحديث عرض صور البوت داخل مقاعد اللعبة وفي مزامنة الحالة وواجهات الغرفة.
- إزالة لاحقة BOT من الاسم المعروض للبوت.
- زيادة صقل بعض عناصر الواجهة: أزرار الطلب، مؤشرات الدور، بطاقات الغرف، صندوق الإشعارات، محاذاة الأزرار، ومعاينة الطاولة.
- رفع إصدار الكاش للأصول إلى v103 لضمان ظهور التعديلات فورًا.
