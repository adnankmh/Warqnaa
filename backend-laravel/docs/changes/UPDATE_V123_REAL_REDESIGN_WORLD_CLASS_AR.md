# تحديث v123 — إعادة تصميم وبرمجة حقيقية

## الهدف
هذه النسخة ليست مجرد إضافة ملفات؛ تم تغيير واجهات أساسية وإعادة بناء صفحة الألعاب والمتجر وسلوك الدردشة وطبقة محركات الألعاب.

## أهم التغييرات

### 1. تصميم جديد واضح
- تغيير جذري في الخلفيات والألوان.
- أخضر مخملي + ذهبي + واجهة تطبيق فخمة.
- تقليل المساحات الفارغة.
- ضبط الصفحة لمنع الطفح والخروج من الإطار.

### 2. صفحة كل الألعاب
- إعادة بناء كاملة لصفحة الألعاب.
- إطار واحد داخل الشاشة.
- Sidebar احترافي.
- بطاقات صغيرة مضغوطة.
- وضع Ultra Dense.
- بحث وفلاتر بدون خروج عن الصفحة.
- لا يوجد horizontal overflow.

### 3. المتجر
- إعادة بناء صفحة المتجر.
- تبويبات علوية.
- بحث.
- فلترة tier.
- بطاقات فخمة.
- معاينات حقيقية للطاولات وظهر الورق والإيموجي والمسرعات والمؤثرات.
- المشتريات ضمن نفس الإطار.

### 4. الدردشة
- رجوع لسلوك v115:
  - إغلاق كامل.
  - أيقونة عودة فقط.
  - لا تبقى ظاهرة بعد الإغلاق.
  - سحب وتصغير.

### 5. محركات الألعاب
- إضافة UniversalSocialGameRules كمحرك آمن لأي لعبة لا تملك محركًا عميقًا.
- كل لعبة في الكتالوج أصبحت تملك محركًا يرجع state صالح.
- تقوية Tarneeb لقبول صيغ متعددة للورق:
  - card
  - card_id
  - rank + suit
  - رموز عربية وإنجليزية للأنواع.

### 6. اختبارات
- إضافة:
  - tests/Feature/V123FinalQualityTest.php
- اختبار أن كل لعبة من الكتالوج ترجع initial state.
- اختبار صيغ مختلفة للطرنيب.
