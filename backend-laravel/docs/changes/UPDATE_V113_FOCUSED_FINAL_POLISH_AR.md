# تحديث v113 — تعديلات مركزة على طلب المستخدم

## تم تنفيذ
1. أيقونات الإشعارات الرئيسية أصبحت تنقلك لصفحة الإشعارات بدل فتح خانة Scroll.
2. تحسين معاينات المتجر والإدارة:
   - الطاولات تظهر كطاولة لعب كاملة.
   - ظهر الورق يظهر كأوراق.
   - الإيموجز تظهر كمعاينة واضحة.
   - المسرعات تظهر كمكوك فضائي 3D.
3. تصنيفات الطاولات وظهر الورق والإيموجز أصبحت أقوى:
   - يتم استنتاج نوع العنصر من key / payload / النص / class.
   - المحترف يظهر تحت احترافي.
   - المبتدئ تحت مبتدئ.
   - المتوسط تحت متوسط.
   - المميز تحت مميز.
   - الأسطوري تحت أسطوري.
   - المتحرك تحت متحركة.
4. الثيم لا يعطي رسالة تفعيل عند تحميل الصفحة.
   - الرسالة تظهر فقط عند ضغط اللاعب على ثيم جديد.
5. الدردشة:
   - تغلق إلى أيقونة أسفل الشاشة.
   - تعود كاملة عند الضغط على الأيقونة.
   - قابلة للسحب من رأس الدردشة.
   - قابلة للتكبير/التصغير اليدوي.
6. زر كل الألعاب يفتح شاشة كاملة فخمة ومنظمة لكل الألعاب.
7. نظام الأصدقاء:
   - إجراءات AJAX للطلب/القبول/الرفض/الإلغاء/الحظر قدر الإمكان.
   - أزرار تتحول بعد تنفيذ العملية.
8. تحسين Responsive وتقليل تداخل الأزرار.
9. المسرعات صارت مكوك فضائي 3D ملون حسب لون كل مسرع.
