# Warqna v137 — لوحة التحكم الشاملة بدون أكواد

هذه النسخة تضيف لوحة إدارة موسعة تسمح للمدير بتغيير شكل الموقع وإدارته من واجهة رسومية دون تعديل الملفات.

## أين تجدها؟
من حساب المدير:
`/admin`

ثم استخدم التبويبات الجديدة:

- **مصمم شامل**: تعديل عرض الأزرار، ارتفاعها، لونها، شكلها، الحواف، البطاقات، الخلفية، الطاولات، المتجر، الدردشة، الإشعارات والبروفايل.
- **الألعاب والقوانين**: تفعيل أو تعطيل الألعاب، تعديل أسماء الألعاب، المحرك، عدد اللاعبين، القوانين بالعربي والإنجليزي و JSON القواعد.
- **إدارة المتجر**: تعديل أسعار عناصر المتجر، مدة الاشتراك، القسم، اللون، الأيقونة، المسرّع، رفع الصور والمعاينة.
- **كل اللاعبين**: تعديل بيانات اللاعبين، XP، اللون، التوكنز، الحظر، الحذف والصداقة.
- **الغرف المفتوحة**: تعديل الحالة، الظهور، كلمة السر، عدد اللاعبين، رسوم الدخول، المستوى وهدف النقاط.
- **النوادي**: تعديل الاسم، المستوى، نقاط الأسبوع والخزينة أو حذف النادي.
- **المنافسات**: إنشاء منافسة جديدة أو تعديل الرسوم والجائزة والحالة والمقاعد والمراحل.

## التخزين
كل إعدادات التصميم محفوظة في جدول `site_settings` وتُطبّق مباشرة عبر CSS Variables داخل `layouts/app.blade.php`.

## Migration مضافة
`database/migrations/2026_07_02_000137_no_code_admin_studio_defaults.php`

تشغّلها كالعادة:

```bash
php artisan migrate
```

## ملاحظات
- لا تحتاج كتابة CSS يدويًا لتغيير الأزرار أو الطاولات أو المتجر.
- حقل `custom_css` القديم بقي موجودًا لمن يريد تخصيصات متقدمة جدًا.
- تم إبقاء التعديلات السابقة للمتجر والدردشة والإشعارات وسجل التوكنز.
