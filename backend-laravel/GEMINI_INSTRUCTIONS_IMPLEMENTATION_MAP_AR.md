# خريطة دمج تعليمات جيميني في v118

تمت قراءة الملف وتحويله إلى خطة تنفيذ عملية داخل مشروع Laravel الحالي.

## المحاور المستخرجة
- هوية بصرية فاخرة: أخضر مخملي + ذهبي ملكي + طاولات وأوراق احترافية.
- واجهات كاملة: الدخول، الرئيسية، اللوبي، طاولة اللعب، المتجر، البروفايل، الأصدقاء، النوادي، المسابقات، الإعدادات.
- محركات ألعاب: عائلات اللمّات، التجميع، العقود، الرقعة/النرد، الدومينو.
- حماية: Server-Authoritative، عدم إرسال أوراق الخصوم، تحقق الحركات، رصد الغش والتواطؤ.
- تجربة اجتماعية: أصدقاء، دردشة، تفاعلات ومقذوفات، نوادي، خزينة، حروب نوادي.
- اقتصاد: توكنز، عملات، متجر، مقتنيات نادرة، مواسم، عروض، إعلانات مكافأة، مكافآت يومية.
- Realtime/WebSocket foundation مع fallback.
- دعم 7 لغات، ثيمات، خطوط، PWA، وتحضير Android/iOS.
