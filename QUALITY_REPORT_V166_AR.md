# تقرير جودة Warqna v166

## العقود التي تم تثبيتها

- مصدر إصدار موحد `1.66.0+166`.
- استمرار حماية WorkManager وعدم تشغيل الإضافات الاختيارية قبل أول إطار.
- صلاحيات Android للصوت والإشعارات والشبكة.
- عدم وجود عبارة `الأراضي الفلسطينية` في المصدر.
- توفر 12 صورة لعبة و18 ملف صوت.
- توفر شاشة اتصال وصوت، رمي سريع، استئناف الغرفة، تقرير نقاط، دردشة موحدة، معاينات المتجر وسياسة الخصوصية.
- تخزين Push tokens مشفرًا في Laravel مع hash فريد.
- مرسل Firebase HTTP v1 آمن يفشل دون تعطيل الدردشة، ويحذف الرموز الملغاة تلقائيًا.
- تسجيل FCM token بعد تسجيل الدخول وتجديده وحذفه عند تسجيل الخروج.

## حدود التحقق المحلي

تم تنفيذ فحوص البنية وPHP وJSON وYAML وShell والحزمة المضغوطة. لا يتوفر Flutter SDK أو Composer vendor في بيئة التجهيز، ولذلك تبقى `flutter analyze` و`flutter test` و`php artisan test` اختبارات GitHub Actions الملزمة.

## إعدادات خارجية مطلوبة

- خادم Laravel منشور عبر HTTPS.
- TURN حقيقي لضمان الصوت بين الشبكات المختلفة.
- مشروع Firebase وبيانات إرسال من الخادم للإشعارات عند إغلاق التطبيق.
- مفاتيح OAuth الفعلية لتسجيل Google/Facebook/Apple.
