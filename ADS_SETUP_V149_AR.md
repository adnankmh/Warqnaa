# إعداد AdMob في v149

أضف في GitHub Repository Variables:

- `ADMOB_ANDROID_APP_ID`
- `ADMOB_REWARDED_ANDROID_ID`
- `ADMOB_IOS_APP_ID`
- `ADMOB_REWARDED_IOS_ID`

ثم شغّل Workflow الخاص بـAndroid أو iOS. عند عدم وضع القيم تُستخدم وحدات الاختبار الرسمية. لا تستخدم وحدات الاختبار عند النشر التجاري.

مكافأة الإعلان في هذه النسخة: 50 توكن + 15 XP، بحد أقصى 5 مرات يوميًا.
